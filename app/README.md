# 💻 app — אפליקציית הפלטפורמה

> יישומוני HTML מרכזיים: אתר קורס ודף נחיתה

## סקירה

מודול האפליקציה מכיל את שני היישומונים הבסיסיים של DJMaster Academy בשלב Phase 0 (HTML5 טהור). אתר הקורס הוא פלטפורמה לימודית מלאה עם 10 שיעורים, חידונים אינטראקטיביים, וניהול התקדמות. דף הנחיתה הוא portal התמרה להצבת מנויים חדשים.

## יישומונים בודול זה

| יישומון | תיאור | קובץ | גודל | סטטוס |
|--------|-------|------|------|-------|
| **DJ Course Site** | 10 שיעורים, חידונים, תעודה דיגיטלית, התקדמות | `dj-course.html` | 3,077 שורות | ✅ |
| **Landing Page** | 9 סקשנים, pricing, testimonials, CTA | `landing-page.html` | 1,681 שורות | ✅ |

## Quick Start

```bash
# קורס
open app/dj-course.html

# דף נחיתה
open app/landing-page.html
```

כל קובץ עובד בעצמאות בדפדפן (HTML5 + CSS + JS).

## מבנה דף קורס

- **10 שיעורים:** World of DJ → Manual Beatmatching + 5 advanced
- **Quizzes:** חידון אינטראקטיבי לאחר כל שיעור
- **Progress:** ניהול התקדמות עם LocalStorage
- **Certificate:** תעודה דיגיטלית בסיום
