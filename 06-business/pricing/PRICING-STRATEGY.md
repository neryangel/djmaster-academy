# Pricing Strategy — DJMaster Academy

## מודל Freemium-to-Pro

### 🆓 Free Tier
**מטרה:** Lead generation, הוכחת ערך

| מה כלול | מגבלה |
|---------|--------|
| 3 שיעורים (Lesson 1–3) | רק וידאו, לא interactive |
| BPM Calculator | מלא |
| Harmonic Wheel | מלא |
| Discord (read-only) | לא יכול לפרסם |
| Email Newsletter | כן |

**הגיון:** הכלים חינמיים → Lead magnet חזק → Upsell לPro

---

### 🟠 Pro Monthly — ₪79/חודש
**מטרה:** Entry level, נגישות מקסימלית

| מה כלול |
|---------|
| כל הקורסים (40+ שיעורים) |
| כלים מתקדמים (Set Planner, Metronome) |
| Discord Full Access |
| Mix Feedback (2/חודש) |
| תעודה דיגיטלית |
| עדכוני תוכן חדשים |

**Positioning:** "פחות מ-₪3 ליום"

---

### 🟠 Pro Yearly — ₪649/שנה
**מטרה:** LTV מקסימלי, הפחתת churn

| מה כלול |
|---------|
| הכל מPro Monthly |
| חיסכון של ₪300 (31%) |
| Bonus: Workshop מוקלט (₪200 ערך) |
| Badge מיוחד בDiscord |

**Anchor Price:** מציג ₪948 (x12) → מוחק → ₪649

---

### 💜 Mentorship — ₪299/חודש
**מטרה:** High-ticket, Revenue per user גבוה

| מה כלול |
|---------|
| הכל מPro |
| Zoom אישי 1×1 (45 דקות/חודש) |
| Feedback מפורט על מיקסים (4/חודש) |
| גישה ל-WhatsApp ישיר |
| Set Review לפני גיגים |
| מקסימום 20 תלמידים במקביל |

**Positioning:** "יש לך מדריך אישי"

---

### 🌟 Lifetime — ₪1,999
**מטרה:** Cash flow מיידי, מי שרוצה הכל

| מה כלול |
|---------|
| גישה לנצח לכל התוכן |
| כל עדכוני העתיד |
| Discord Lifetime Badge |
| 1 Zoom consultation בחינם |

**מתי להציע:** Launch events, מבצעי Black Friday

## 🎯 Pricing Psychology

### Anchoring
```
❌ Pro Monthly ₪79
✓ [מוצג ראשון] Mentorship ₪299 | Pro Yearly ₪649 | Pro Monthly ₪79
```

### Urgency (Launch)
- "המחיר עולה ב-20% ביום שישי"
- "נשארו 50 מקומות בMentorship"

### Value Framing
- "שיעור ג'יטרה → ₪300. DJMaster 3 חודשים → ₪237. החלטה פשוטה."

### Social Proof
- "347 DJs ישראלים כבר למדו השבוע"
