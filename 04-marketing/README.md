# 📧 04-marketing — שיווק ותוכן

> רצף דוא״ל ברכישה, לוח תוכן לרשתות חברתיות, ותבניות קידום

## סקירה

מודול השיווק מכיל את כלי ההמרה (conversion) והאנגייג'מנט (engagement). רצף דוא״ל ברכישה מוציא ערך לתלמידים חדשים, ולוח התוכן מספק מנות יומיות/שבועיות לטיקטוק, אינסטגרם ולינקדין. הכל בעברית וממוקד בקהל DJ ישראלי.
