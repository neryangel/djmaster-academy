1. WAVEFORM VISUALIZER
   File: src/components/audio/WaveformVisualizer.tsx
   Lines: 334
   Purpose: Professional audio waveform visualization with real-time playback
   
   Features:
   - Canvas API real-time waveform rendering
   - Draggable playhead for seeking (Pink #FF6584)
   - Zoom controls (0.5x to 4x magnification)
   - Frequency spectrogram below waveform
   - Time markers and duration display
   - Responsive design with RTL support
   - Grid overlay for visual reference
   
   Key Props:
   • audioUrl: string (required)
   • playerState: AudioPlayerState (required)
   • onSeek?: (time: number) => void (optional)
   • height?: number (optional, default: 200)
   • showSpectrogram?: boolean (optional, default: true)
   • showTimeMarkers?: boolean (optional, default: true)

2. MIDI CONTROLLER UI
   File: src/components/midi/MIDIControllerUI.tsx
   Lines: 503
   Purpose: Interactive visual representation of Pioneer DDJ-FLX4 controller
   
   Features:
   - Two virtual jog wheels (Deck A & B) with SVG rotation
   - EQ knobs (Hi/Mid/Lo) with rotary SVG visualization
   - Volume faders for each deck
   - Play/Cue/Sync buttons with LED-style indicators
   - 8 performance pads per deck (4x2 grid) with color states
   - Visual crossfader slider between decks
   - VU meter displays for volume levels
   - MIDI connection status indicator
   - All labels in Hebrew
   
   Key Props:
   • connectionState: MidiConnectionState (required)
   • onMidiEvent?: (event: MidiEvent) => void (optional)
   • decksState?: DeckState (optional)
   • activePads?: { A: number[]; B: number[] } (optional)
   • crossfaderPosition?: number (optional)

3. ACHIEVEMENT TOAST
   File: src/components/ui/AchievementToast.tsx
   Lines: 255
   Purpose: Notification toast for achievements with animated effects
   
   Features:
   - Slides in from top-right (top-left in RTL)
   - Achievement icon, name (Hebrew), XP reward display
   - Animated confetti particle system
   - Auto-dismisses after 5 seconds with progress bar
   - Stacks multiple toasts automatically
   - Rarity-based styling (Common/Uncommon/Rare/Epic/Legendary)
   - Color scheme: Gold for legendary, Purple for rare, Cyan for uncommon
   - Close button for manual dismissal
   - Smooth fade-out animation
   
   Key Props:
   • badge: Badge (required)
   • onDismiss?: () => void (optional)
   • duration?: number (optional, default: 5000ms)
   • stackPosition?: number (optional, default: 0)
   
   Sub-component:
   • AchievementToastContainer: Manages multiple toasts

4. LESSON NAVIGATOR
   File: src/components/course/LessonNavigator.tsx
   Lines: 316
   Purpose: Sidebar lesson navigator with expandable modules
   
   Features:
   - Accordion-style module expansion
   - Lesson completion checkmarks
   - Lesson type indicators (Video/Text/Quiz/Exercise)
   - Duration display per lesson
   - Lock icons for inaccessible lessons
   - Module-level progress bars
   - Overall course progress bar
   - Sticky sidebar behavior
   - Mobile-responsive collapsible drawer
   - All Hebrew text
   - Visual highlighting of current lesson
   
   Key Props:
   • modules: Module[] (required)
   • currentLessonId?: string (optional)
   • currentModuleId?: string (optional)
   • onLessonSelect?: (lessonId, moduleId) => void (optional)
   • courseProgress?: number (optional, default: 0)
   • isCompact?: boolean (optional, default: false)
