# 📚 03-assets — חומרי עזר ומשאבים

> מילון, כרטיסי עזר מהיר וחומרים רפרנס לתלמידים

## סקירה

מודול זה מכיל את חומרי העזר המשלימים את הקורסים. קלוט מהיר למונחים בדו"ח, כרטיסי עזר (cheatsheets) שניתן להדפיס, וצפיפוטים לשיטות ערבוב (mixing). כל חומר מעוצב לחזרה מהירה בזמן ביצוע (במהלך set או בהכנה).

## קבצים בודול זה

| קובץ | תיאור | פורמט |
|------|-------|-------|
| **glossary/DJ-GLOSSARY.md** | מילון מונחים בדו"ח עברית ↔ אנגלית (200+) | Markdown |
| **cheatsheets/FLX4-QUICK-REF.md** | טבלת בקרה DDJ-FLX4 — לחצנים, פאדים, rotary | Markdown |
| **cheatsheets/MIXING-RULES.md** | 15 כללי ערבוב חיוניים (transition, EQ, levels) | Markdown |
| **cheatsheets/flx4-cheatsheet.pdf** | גרפיקה מודפסת של FLX4 עם תיוגים | PDF |

## Quick Start

1. **מונחים לא ברורים?** → `glossary/DJ-GLOSSARY.md`
2. **שחקתי בFLX4, מה זה שם?** → `cheatsheets/FLX4-QUICK-REF.md`
3. **איך מערבבים נכון?** → `cheatsheets/MIXING-RULES.md`
4. **הדפסה לטבלה?** → `cheatsheets/flx4-cheatsheet.pdf`
