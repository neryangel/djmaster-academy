# DDJ-FLX4 — Quick Reference Card

## 🎛️ מיפוי כפתורים עיקריים

| כפתור / ידית | פונקציה |
|--------------|---------|
| **TEMPO Fader** | שינוי מהירות (BPM ±8%) |
| **PITCH BEND ±** | דחיפת BPM זמנית |
| **SYNC** | סנכרון BPM אוטומטי |
| **CUE** | נקודת התחלה (Hot Cue A) |
| **PLAY/PAUSE** | הפעלה / השהיה |
| **JOG WHEEL** | מניפולציה ידנית / Scratch |
| **VINYL MODE** | מצב וינייל ל-Jog |
| **BEAT LOOP** | לולאה על Beat מוגדרת |
| **HOT CUE 1–4** | נקודות Hot Cue מהירות |
| **CH FADER** | Volume ערוץ |
| **CROSSFADER** | מעבר בין Deck A↔B |
| **TRIM** | עוצמת כניסה (Gain) |
| **EQ HI/MID/LOW** | אקולייזר 3 פסים |
| **FILTER** | פילטר Hi/Lo-Pass |
| **HEADPHONE CUE** | האזנה פרטית לערוץ |

---

## 🎚️ רצף מיקסינג בסיסי

```
1. טען שיר לDeck A (מנגן)
2. טען שיר לDeck B (הבא)
3. האזן לDeck B בהדפונים
4. כוון BPM (SYNC / Tempo Fader)
5. כוון פאזה (Jog Wheel)
6. כוון EQ (הנמך LOW ב-B)
7. הוצא CH Fader B לאט
8. משוך Crossfader מA לB
9. הכנס LOW ב-B, הנמך ב-A
10. הורד CH Fader A לגמרי
```

## 🔄 Hot Cue — שימוש מהיר

| פעולה | איך |
|-------|-----|
| **שמור Hot Cue** | לחץ HOT CUE בזמן נגינה |
| **קפוץ ל-Hot Cue** | לחץ שוב על אותו כפתור |
| **מחק Hot Cue** | SHIFT + HOT CUE |
| **סנכרן ל-Hot Cue** | SHIFT + PLAY בצד השני |

## 🌀 Beat Loop מהיר

| גודל לולאה | כפתור |
|-----------|-------|
| 1/2 Beat | LOOP 1/2 |
| 1 Beat | LOOP 1 |
| 2 Beats | LOOP 2 |
| 4 Beats | LOOP 4 |
| 8 Beats | LOOP 8 |

> 💡 **טיפ:** שחרר לולאה בלחיצה שנייה על אותו כפתור

## ⌨️ Rekordbox — קיצורי מקלדת

| מקש | פעולה |
|-----|-------|

| `1–4` | Hot Cues |

| `Ctrl+L` | Load לDeck |
| `Ctrl+M` | Analyze Track |
