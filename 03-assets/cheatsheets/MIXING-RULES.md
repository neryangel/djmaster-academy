# חוקי מיקסינג — The Golden Rules

## ✅ The 5 Laws of Mixing

### 1. BPM — תמיד סנכרן לפני
- הפרש מקסימלי בין שירים: **±6 BPM**
- מעל 6 BPM = השמע ניגוד, לא מיקס

### 2. EQ Swap — חוק ה-3 פסים
```
כשמוציא שיר חדש:
  B-LOW = OFF (עד שA נגמר)
  B-MID = 80%
  B-HI  = 100%

כשמסיר שיר ישן:
  A-LOW = OFF → ON ל-B
  A-MID = 50% → OFF
```

### 3. Phrase Matching — 8 Beats כלל
- כניסה תמיד ב-downbeat (beat 1)
- מעברים בין **32 או 64 beats**
- לעולם לא ditching באמצע phrase

### 4. Energy Arc — בניית סט
```
דקות 0–20:  Warm Up (BPM 120–125)
דקות 20–50: Build Up (BPM 126–130)
דקות 50–70: PEAK (BPM 128–132)
דקות 70–90: Come Down (BPM 124–126)
```

### 5. Harmonic — Camelot לפני הכל
- מיקס בין A↔A, B↔B, או +1/-1 = בטוח
- מיקס חוצה שתי קטגוריות = רועש
- פתח Rekordbox → Key Analysis לפני כל סט

## 🚫 שגיאות נפוצות למתחילים

| שגיאה | מה קורה | פתרון |
|-------|---------|--------|
| BPM לא מסונכרן | "Train wreck" | SYNC לפני כניסה |
| כניסה מוקדם מדי | חפיפה מוזרה | חכה ל-phrase boundary |
| שני LOW-ים יחד | BOOM כבד | כבה LOW בשיר הנכנס |
| לא מאזין להדפונים | הפתעות רעות | CUE תמיד לפני |
| מיקס ז'אנרים שונים | אי-נוחות לקהל | עקביות ז'אנר לפחות 3 שירים |
