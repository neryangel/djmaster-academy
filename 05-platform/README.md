# 🎨 05-platform — עיצוב ופיתוח

> Design System, Tech Stack וארכיטקטורה Front-end

## סקירה

מודול הפלטפורמה מכיל את שלד ה-UI/UX של DJMaster Academy. Design System מגדיר צבעים, טיפוגרפיה וקומפוננטים; Tech Stack מתעד את הכלים להוצאה לפועל בשלבים עתידיים. זוהי התשתית לפיתוח Next.js + Supabase.

## קבצים בודול זה

| קובץ | תיאור | תכולה |
|------|-------|-------|
| **design/DESIGN-SYSTEM.md** | צבעים, טיפוגרפיה, breakpoints, קומפוננטים | CSS variables, גרידים, סימנים |
| **tech/TECH-STACK.md** | בחירות טכנולוגיה: Frontend, Backend, DevOps | סיבות בחירה, alternatives |

## Quick Start

1. **עיצוב:** קרא `design/DESIGN-SYSTEM.md` לפני כתיבת קוד חדש
2. **תכנות:** קרא `tech/TECH-STACK.md` כדי להבין את infrastructure
