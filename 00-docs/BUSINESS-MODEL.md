# מודל עסקי — DJMaster Academy

## תקציר ניהולי

**DJMaster Academy** היא פלטפורמת החינוך המקיפה בעברית לדי-ג'יים, המתמקדת בהכשרה מאפס עד מקצוע. אנו מציעים שילוב ייחודי של תוכן בעברית באיכות גבוהה, כלים אינטראקטיביים, וקשרים עם קהילת ה-DJ הישראלית.

**שנה ראשונה (Y1):** יעד של 1,000 מנויים פעילים, הכנסה של ₪948,000 ברוטו.
**שנים 2-3:** בהנחת צמיחה של 200-300%, להגיע ל-3,000 מנויים (Y2) ו-8,000 מנויים (Y3).

## הערך המוצע (Value Proposition)

### למה DJMaster Academy?

1. **תוכן בעברית מלאה** — כל הקורסים, ההנחיות וחומרי העזר בעברית ברורה ומקצועית
2. **התמחות ב-FLX4** — ה-controller היחיד שמלמדים באופן מעמיק עם Rekordbox Pro
3. **מנטורים מנוסים** — DJ-ים ישראליים בעלי ניסיון 10+ שנים בבמה ובמועדונים
4. **כלים אינטראקטיביים** — BPM Calculator, Harmonic Wheel, Set Planner, כולם עוזרים בתרגול בזמן אמת
5. **קשר קהילתי חזק** — Discord פעיל, פורום, תחרויות בחודשון (mix submissions)
6. **צמוד אל צרכים המקומיים** — הדרכה על חתונות, אירועים ישראליים, ז'אנרים פופולריים בישראל

## אסטרטגיית מחירים (Pricing Strategy)

### ייחודיות של מודל הטיפול:

1. **Free Tier** — מצנן שער (Freemium)
   - מינימום של תוכן לאימות הערך

   - מהדדות מתפרסמות חודשיות בשטיח הצלחה

3. **Mentorship**
   - פרמיום 3.8x ביחס Pro Monthly

4. **Lifetime**
   - פרמיום עד ₪2,000 (וחצי מחירו של Mentorship שנתי)
   - מודל שנקי למי שסופיים עם הצעדים היחידים

## מבנה עלויות וניתוח איזון (Cost Structure & Break-even)

### עלויות קבועות (Fixed Costs) — חודשיות:

| פריט | עלות חודשית |
|------|-----------|
| אחסון וקידום (AWS, CDN) | ₪3,000–5,000 |
| פתיחת תוכן (עריכה וידאו, עיצוב) | ₪8,000–10,000 |
| שכר (מנהל + תומך קהילה) | ₪12,000–15,000 |
| כלים (Slack, Figma, יעדות בחינם) | ₪1,000–2,000 |
| שיווק (Google Ads, Tiktok Ads) | ₪5,000–10,000 |
| משכנתא משרד + כלים | ₪2,000–3,000 |
| **סך קבוע בחודש** | **₪31K–45K** |
| **סך קבוע בשנה** | **₪372K–540K** |

### עלויות משתנות (Variable Costs):

| פריט | עלות |
|------|------|
| Zoom (Mentorship) | ₪5 לשעה |
| Merchandise (לפי הזמנה) | 40-50% מהמחיר |
| Affiliate (על מכירות) | 0% עלות (מוגדל בהכנסה) |

### איזון ספיר (Break-even Analysis) — Y1:

```
עלויות קבועות בחודשון = ₪38K (ממוצע)
עלויות קבועות בשנה = ₪456K

יעד הכנסה ברוטה שנתי = ₪948K
עלויות משתנות = ₪100K (Merchandise, Affiliate)
הכנסה נטו שנתית = ₪948K – ₪456K – ₪100K = ₪392K

Break-even Point (BEP):
BEP = עלויות קבועות / שולים גולמיים
BEP = ₪456K / 60% = ₪760K בהכנסה ברוטה

בהנחה:
- ממוצע הכנסה חודשית = ₪79K
- זמן למגע שבוער = 6-7 חודשים

בחודש 7: בריאות (צמיחה תמידית)
```
