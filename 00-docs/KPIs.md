## KPIs גדילה (Growth KPIs)

### User Acquisition (רכישת משתמשים):

| ערוץ | Y1 יעד | Cost Per User | כמות משוערת |
|------|-------|---------------|-----------|
| YouTube/SEO (אורגני) | 300 users | ₪0 | 300 |
| TikTok/Social | 200 users | ₪75 | 200 |
| Google Ads | 150 users | ₪150 | 150 |
| Referral (Word of Mouth) | 150 users | ₪50 | 150 |
| Partnerships | 100 users | ₪100 | 100 |
| **סך הכל** | **900 users** | **~₪80 ממוצע** | — |

### CAC (Cost of Acquisition):

```
סכום שהוצאה לשיווק (Y1) = ₪72,000
משתמשים חדשים (Y1) = 900
CAC Y1 = ₪72,000 / 900 = ₪80

יעד: <₪150 (מושג ✅)
```

### LTV (Lifetime Value):

```
Pro Monthly (משוקלל):
- Average Lifetime = 10 חודשים
- ARPU = ₪79
- LTV = ₪79 × 10 = ₪790

Mentorship (משוקלל):
- Average Lifetime = 6 חודשים
- ARPU = ₪299
- LTV = ₪299 × 6 = ₪1,794

LTV ממוצע (weighted) = ₪850–900 בשנה
יעד: >₪800 (מושג ✅)
```

### LTV:CAC Ratio:

```
LTV:CAC = ₪850 / ₪80 = 10.6x

יעד: >5x בריא (מושג ✅ עם מרווח טוב)
```

### Payback Period:

```
Payback = CAC / (ARPU × Gross Margin%)
Payback = ₪80 / (₪79 × 85%)
Payback = ₪80 / ₪67
Payback = 1.2 חודשים

יעד: <3 חודשים (מושג ✅ יותר מטוב)
```

### Viral Coefficient:

```
כל משתמש חדש מביא בממוצע 0.3 משתמשים נוספים
(דרך Referral, Word of Mouth, Share בפורום)

יעד Y1: 0.2–0.3 (בתור עיסוק)
יעד Y2: 0.4–0.5 (במידי גדל)
```

## סדר בדיקה רבעוני (Quarterly Review Cadence)

### מהלך הבדיקה הרבעונית:

**שבוע 1 של Q+1 (לדוגמה: 1/1/2025):**

1. **Data Collection** (יום 1–2)
   - Pull all KPI data from analytics
   - Organize in Google Sheet or Dashboard

2. **Analysis** (יום 3–4)
   - Compare to targets
   - Identify wins and gaps
   - Analyze root causes

3. **Team Sync** (יום 5 — 1 שעה)
   - Present findings
   - Discuss challenges
   - Set focus areas for Q+1

4. **Action Planning** (יום 6–7)
   - Create OKRs for Q+1
   - Assign owners
   - Set priorities

### KPIs to Review by Role:

**Product:**
- Lesson Completion Rate
- Feature Adoption
- Churn Rate

**Marketing:**
- CAC by Channel
- Email Open Rates
- Social Engagement

**Business:**
- MRR / ARR
- LTV:CAC Ratio
- Payback Period

**Community:**
- Discord Members
- Mix Submissions
- NPS Score
