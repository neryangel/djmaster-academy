## 📖 מבנה כל שיעור (Course Structure)

כל קורס מכיל:

```
01-world-of-dj/
├─ lesson.md          ← תסריט + תוכן עיקרי (250+ שורות)
├─ exercises.md       ← 5–6 תרגילים עם רובריקה
├─ slides.pptx        ← מצגת ויזואלית (26–40 slides)
├─ quiz.json          ← 10–15 שאלות בחינה (סגור וגלוי)
├─ resources.md       ← קישורים + וידאוים + מילון
└─ README.md          ← סקירה קצרה

סה"כ זמן לימוד: שיעור (50 דק) + תרגילים (45–60 דק) + קוויז (20 דק)
זמן כולל: 2–2.5 שעות למטל
```

## 📊 מאתר לימודי (Learning Path)

### עבור מתחיל (חודשיים):

```
שבוע 1–2: Lessons 01–02 (Theory + Equipment)
          ↓
שבוע 3–4: Lessons 03–04 (Software + Music Structure)
          ↓
שבוע 5–8: Lesson 05 (Manual Beatmatching — practiced!)
          ↓
בסוף: יכול להנגן 30 דקות מיקס בלי טעויות
```

### עבור תיכון (3–6 חודשים):

```
חודש 1–2: Lessons 01–05 + Lesson 06 (EQ)
          ↓
חודש 3–4: Lessons 07–08 (Pads & Effects)
          ↓
חודש 5–6: Lesson 09–10 (Set Building & Live)
          ↓
בסוף: יכול ללנגן 1–2 שעות במועדון / ביתי
```

### עבור מתקדם (6+ חודשים):

```
חודש 1–2: Lessons 01–10 (בדיקה מהירה)
          ↓
חודש 3–6: 2–3 Masterclasses (בחר genre)
          ↓
בסוף: DJ מקצוע בשלם
```
