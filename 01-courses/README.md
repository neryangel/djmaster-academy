# 🎓 01-courses — תוכן לימודי

> 5 קורסים בסיסיים (40+ שיעורים בתוכנית המלאה) עם תרגילים, שאלות ותעודות

## סקירה

מודול התוכן הלימודי מכיל את הליבה של DJMaster Academy. כל קורס מורכב משיעור מנויין, תרגילים מעשיים, וחידון אינטראקטיבי. במבנה זה אפשר להרחיב ל-40+ שיעורים נוספים כמתוכנן.

## קורסים בסדר ההפעלה

| # | קורס | תיאור | קבצים |
|---|------|-------|-------|
| 1 | **World of DJ** | היסודות: כרטוג'ים, פורמט אודיו, תיאוריה בסיס | `lesson.md` + `exercises.md` + `quiz.json` |
| 2 | **FLX4 Equipment** | התחנה: DDJ-FLX4, לחצנים, פאדים, טופוגרפיה | `lesson.md` + `exercises.md` + `quiz.json` |
| 3 | **Rekordbox** | תוכנת הניהול: ספריה, cueing, looping, effects | `lesson.md` + `exercises.md` + `quiz.json` |
| 4 | **Music Structure** | תיאוריה מתקדמת: phrasing, intro/outro, harmonic key | `lesson.md` + `exercises.md` + `quiz.json` |
| 5 | **Manual Beatmatching** | טכניקה ידנית: BPM, pitch, jog wheel control | `lesson.md` + `exercises.md` + `quiz.json` |

## Quick Start

```bash
# קרא את המפה
COURSES-MAP.md

# תחילת קורס 1
cd 01-world-of-dj/
cat lesson.md       # השיעור
cat exercises.md    # תרגילים
cat quiz.json       # שאלות

# המשך בקורס 2, 3, 4, 5
```

## מבנה קורס (template)

כל תיקייה מכילה:
- `lesson.md` — תוכן שיעורי עם דוגמאות
- `exercises.md` — 3-5 תרגילים מעשיים
- `quiz.json` — שאלון בחירה מרובה (5-10 שאלות)
