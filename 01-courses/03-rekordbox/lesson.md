## 1. התקנה והגדרות ראשוניות

### שלב 1: הורדה והתקנה

**איפה להוריד:**
- https://rekordbox.com (הגירסה החופשית או Premium)

**מה להוריד:**
- **Rekordbox (Free)** — סביר לחלוטין להתחלה
- **Rekordbox (Paid/Premium)** — כוללת יותר תכונות

**שלבי התקנה (Windows/Mac):**
1. הורד את ה-Installer
2. פתח את ה-Installer
3. בחר מקום ההתקנה
4. המתן 2-3 דקות
5. הפעל את Rekordbox

### שלב 2: הגדרה ראשונית

**התחבר לחשבון Pioneer:**
1. פתח את Rekordbox
2. לחץ על "Sign In"
3. בחר "Create New Account" או "Sign In"
4. השתמש בדוא"ל שלך וצור סיסמה חזקה

**בחר את תיקיית המוזיקה:**
1. Settings → Music Folder
2. בחר את התיקייה שבה יש לך מוזיקה (MP3, FLAC, WAV)
3. Rekordbox תחפש ותייבא את כל הקבצים

**בדוק את הציוד:**
1. Settings → Device Settings
2. בחר DDJ-FLX4 כ-Output
3. בדוק שקול יוצא מהרמקול

## 2. ייבוא מוזיקה וניתוח אוטומטי

### מה זה BPM Analysis?

Rekordbox מנתחת כל שיר ומחשבת:
- **BPM** (Beats Per Minute) — מהירות הטראק
- **Beat Grid** — כאן בדיוק הבעיות של הטראק
- **Waveform** — תצוגה ויזואלית של הטראק

### תהליך הייבוא

**שלב 1: עדכן את תיקיית המוזיקה**
1. File → Music Folder
2. בחר את התיקייה שלך
3. לחץ על "Update" או "Scan"

**שלב 2: המתן לניתוח**
- Rekordbox תמתנה כ-20-30 שניות למאה שיר
- תראה כן-כן בעליון כשהניתוח בעיצומו
- **תן לה לסיים לפני שתשחק!**

**שלב 3: בדוק את התוצאות**
1. תבט על רשימת השירים
2. בדוק שכל שיר יש BPM (למשל, 128 BPM)
3. אם חסר BPM — לחץ במשך ימין ובחר "Analyze" שוב

## 3. ניהול ספריה — Playlists, Tags, Rating

### A. Playlists — ארגון לפי ז'אנר

**מה זה:**
- קלוט של שירים למטרה מסוימת
- דומה ל-Playlist ב-Spotify, אבל לביצוע

**איך ליצור Playlist:**
1. עדיפות שמאל → "Playlist"
2. לחץ על "+" (ליד Playlist)
3. כתוב שם: "House 128 BPM" או "Techno Night"
4. לחץ "Create"

**איך להוסיף שירים:**
1. גרור את השיר מהרשימה הראשית ל-Playlist
2. או בחר את השיר, לחץ כן-כן, ובחר "Add to Playlist"

### B. Tags ו-Color — אודיו מהיר

**מה זה:**
- תגים לעזרה חיפוש מהיר (למשל, "Summer", "Night", "English")
- צבעים לארגון חזותי

**איך להוסיף Tag:**
1. לחץ עליון משך עם עכבר ימני על שיר
2. בחר "Edit → Tags"
3. כתוב תג: "Summer" או "Party"
4. לחץ "Save"

**דוגמה Tags:**
- **Tempo:** "120", "128", "140"
- **Mood:** "Energy", "Chill", "Night"
- **Season:** "Summer", "Winter"

### C. Rating (דירוג) — לבחירה מהירה

**איך לדרג:**
1. לחץ עליון כן-כן על שיר
2. בחר "Rating → ★★★★★" (5 כוכבים)

## 4. Grid Editing — תיקון Beat Grid

### מה זה Beat Grid?

**Rekordbox חשבה את ה-BPM של הטראק, אבל היא לא תמיד צודקת!**

- **Beat Grid** = קו אדום שחוזר על כל beat בטראק
- אם ה-Grid נמצא במקום לא נכון, ה-Beatmatching יהיה מכשול

### איך לתקן את ה-Grid:

**שלב 1: פתח את Grid Editor**
1. לחץ עליון משך עם עכבר ימני על שיר
2. בחר "Edit → Open in Grid Editor"

**שלב 2: תנגן את השיר**
- גמע בקלט Intro ובחפש את ה-Kick Drum הראשון

**שלב 3: עדכן BPM**
- השערה ערך BPM (למשל, 128.0 עד 128.3)
- תנגן 30 שניות
- בדוק: האם הקווים הם בדיוק על כל Kick?

**שלב 4: עדכן את Downbeat**
- הקו הראשון צריך להיות בדיוק על ה-Kick הראשון
- אם הוא מעט או שמאל = גרור לכיוונו הנכון

## 5. Hot Cues & Memory Points

### A. Hot Cues (שמרו בתוכנה)

**מה זה:**
- סימנים בטראק שאתה קובע ברקורדבוקס
- הם נשמרים עם הטראק

**איך להגדיר:**
1. תנגן את הטראק
2. לחץ על "Hot Cue" (כפתור בממשק)
3. שם את הנקודה באופטי אותה רגע

**דוגמה:**
- Hot Cue 1: Intro Start (0:00)
- Hot Cue 2: Verse Begin (0:30)
- Hot Cue 3: Chorus Start (1:00)
- Hot Cue 4: Bridge (1:45)
- Hot Cue 5: Outro (2:30)

### B. Memory Points (יותר מתקדם)

**מה זה:**
- "Memory" של מצב מוזיקלי ספציפי בטראק
- לא רק נקודה, אלא מצב (Cue Point + Effects)

## 6. הגדרת Performance Mode

### מה זה Performance Mode?

**Rekordbox יש שני מודים:**
1. **Export Mode** — ארגון המוזיקה
2. **Performance Mode** — ממשק ביצוע עם בקר

### איך להחליף ל-Performance Mode:

1. הקישור דיסק Rekordbox
2. לחץ על "Performance Mode" (או Ctrl+P)
3. בחר "DDJ-FLX4" כ-Deck
4. לחץ "OK"

**כעת אתה רואה:**
- Jog Wheels ב-Rekordbox
- Pads (Hot Cues, Loops, וכו')
- Mixer (EQ, Crossfader)
- Waveform (תצוגה של הטראק)

---

## 7. Export Mode vs Performance Mode

### השוואה:

| יכולת | Export Mode | Performance Mode |
|-------|-------------|------------------|
| **Playlist ניהול** | ✓ | - |
| **Grid Editing** | ✓ | - |
| **דירוג & Tags** | ✓ | - |
| **ביצוע** | - | ✓ |
| **Waveform** | לא | ✓ |
| **DJ Mixing** | לא | ✓ |

## 8. הגדרות Rekordbox לביצוע עם DDJ-FLX4

### שלב 1: בחר את Output

**Preferences → Audio**
1. **Primary Output:** DDJ-FLX4 (MASTER)
2. **Headphones Output:** DDJ-FLX4 (HEADPHONES)

### שלב 2: בדוק את הקישוריות

**Performance Mode:**
1. הפעל Rekordbox (Performance Mode)
2. חבר DDJ-FLX4 ב-USB
3. בדוק: האם הבקר זוהה?

### שלב 3: בדוק את הצלילים

1. תנגן שיר
2. תנגן בבקר (סוב על Jog Wheel)
3. אתה צריך לשמוע קול מהרמקול
