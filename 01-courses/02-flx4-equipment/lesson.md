## 1. סקירת DDJ-FLX4 — מבנה כללי

ה-DDJ-FLX4 הוא בקר DJ ממדורג (2 Deck) עם מיקסר מובנה. הוא מתחלק ל-4 אזורים עיקריים:

### דיאגרמה של הבקר:

```
┌────────────────────────────────────────────────────┐
│          PIONEER DDJ-FLX4 — TOP VIEW              │
├────────────────────────────────────────────────────┤
│                                                    │
│  [DECK 1]        [MIXER]        [DECK 2]         │
│  ┌─────────┐   ┌──────────┐   ┌─────────┐       │
│  │ Jog Whl │   │ Faders   │   │ Jog Whl │       │
│  │  (L)    │   │   EQ     │   │  (R)    │       │
│  │         │   │  Trim    │   │         │       │
│  └─────────┘   └──────────┘   └─────────┘       │
│                                                    │
│  ┌─────────┐   ┌──────────┐   ┌─────────┐       │
│  │  Pads   │   │  Buttons │   │  Pads   │       │
│  │  8x2    │   │  Volume  │   │  8x2    │       │
│  │         │   │  Cue     │   │         │       │
│  └─────────┘   └──────────┘   └─────────┘       │
│                                                    │
│  [USB] [AUX] [HEADPHONES] [MASTER]              │
│                                                    │
└────────────────────────────────────────────────────┘
```

### 4 אזורים עיקריים:

1. **Deck 1 (שמאל)** — בקרה על הטראק הראשון
2. **Deck 2 (ימין)** — בקרה על הטראק השני
3. **Mixer Section (אמצע)** — ערבוב בין הטרקים
4. **Buttons & Controls (תחתון)** — בקרים כללים

## 2. Jog Wheels — שימוש ופונקציות

### מה זה Jog Wheel?

Jog Wheel הוא גלגל מסתובב שמשמש לשתי מטרות:

1. **Platter Mode** — שליטה בהשמעת הטראק
2. **Pitch Mode** — שינוי המהירות של הטראק

### שימוש בـ Jog Wheel:

#### A. **Vinyl Mode (מצב ויניל)**
- גרור את האצבע על הגלגל = תנועה של הטראק
- כמו שיוניות ישנות — בקרה גרביטית טבעית
- **כשכוון Sync מופעל:** יש התנגדות מוכללת (Static Platter)
- **כשכוון Sync כבוי:** גלגל חופשי לחלוטין

#### B. **CDJ Mode (מצב כתום) — CHOOSE**
- גרור למעלה = קדימה מהר
- גרור למטה = אחורה מהר
- בקרה ליניארית, פחות אינטואיטיבית
- יותר מתאים ל-scratching מדויק

#### C. **Pitch Bend**
- כשאתה גורר במהירות = bend זמני
- משחרר = חוזר למהירות נורמלית
- שימוש: לתיקון beat מהיר או אפקט קריאטיבי

### טיפים Pro:
- **Vinyl Mode הוא הברירת המחדל** — התחל בזה
- גרור במסוף החיצוני (edge) לבקרה טובה יותר
- ברפורמנס — תוף קצת על הגלגל כדי להשמיע צליל מעניין

## 3. Performance Pads — Hot Cue, Loop, Slicer, Sampler

ה-Pads הם 8 כפתורים בצבע שונה בכל Deck. לכל Pad יש 4 מודים שונים.

### Pad Modes:

```
┌─────────────────────────────┐
│  DECK PADS (8 Buttons)     │
├─────────────────────────────┤
│  [1] [2] [3] [4]           │
│  [5] [6] [7] [8]           │
│                             │
│  Mode: HOT CUE / LOOP /    │
│  SLICER / SAMPLER          │
└─────────────────────────────┘
```

### A. **Hot Cue Mode** — סימנים מהירים

**מה זה:**
- סימן נקודה ספציפית בשיר
- לחיצה = קפיצה לנקודה זו

**איך להשתמש:**
1. נגן טראק, עצור בנקודה טובה (למשל, תחילת Verse)
2. לחץ על Pad 1 (מוגדר כברירת מחדל ל-Hot Cue)
3. כעת Pad 1 = קפיצה מיידית לנקודה זו
4. אתה יכול להגדיר עד 8 Hot Cues בטראק אחד

**שימוש בביצוע:**
- לחיצה קצרה = קפיצה לנקודה
- לחיצה ממושכת = החזקה בנקודה וכבוי אחרי שחרור

**דוגמה:** שיר בן 3 דקות:
- Pad 1: Intro (0:00)
- Pad 2: Verse 1 (0:30)
- Pad 3: Chorus (1:00)
- Pad 4: Bridge (1:45)
- Pad 5: Outro (2:30)

### B. **Loop Mode** — לולאות חזרתיות

**מה זה:**
- תקוע חלק מהשיר בחזרה אינסופית
- מעניין לעריכת בעלות וייצור אנרגיה

**איו להשתמש:**
1. לחץ על Pad 1 = הפעל Automatic Loop (בדרך כלל 8 bars)
2. לחץ שוב = בטל את ה-Loop
3. לחץ על Pad 2 = הגדל Loop (16 bars)
4. לחץ על Pad 3 = הקטן Loop (4 bars)

**שימוש בביצוע:**
- Loop על ה-Chorus = חוזרים פי 3 על אותו חלק
- Shrink ו-Expand Loop בזמן אמת לאפקט dynamic

### C. **Slicer Mode** — פרוסות קצובות

**מה זה:**
- חלוקת קטע של 8 beats ל-8 חתיכות
- הנגנים בסדר או בעלויות אקראיות

**איך להשתמש:**
1. לחץ על Pad 1 (הפעל Slicer)
2. הטראק מחולק לשמונה חלקים שווים
3. עכשיו כל Pad = מקטעת אחרת
4. לחץ על Pads בסדר או בחזזרות אקראיות

**דוגמה מעשית:**
- Slicer על Vocal Hook = מנגנן קולות מעניינים

### D. **Sampler Mode** — דגימות מעשיות

**מה זה:**
- הקלט קול, אפקט, או משהו אחר
- הנגנו מחדש בלחיצה

**איך להשתמש:**
1. הקלט דגימה (למשל, קריאה של "YEAH!")
2. הודים נשמרת בـ Pad
3. לחץ בזמן ביצוע = נגן דגימה

**דוגמה בביצוע:**
- הקלט צליל של קהל וקצב
- לחץ בביצוע כדי להוסיף אנרגיה

## 4. Mixer Section — פיידרים, EQ, Trim

### פתח הפרק של כל Deck:

```
┌──────────────────────────┐
│    MIXER SECTION         │
├──────────────────────────┤
│  [TRIM] [TRIM]           │
│   L        R             │
│                          │
│  [EQ HIGH] [EQ HIGH]     │
│  [EQ MID]  [EQ MID]      │
│  [EQ LOW]  [EQ LOW]      │
│                          │
│    [CROSSFADER]          │
│    [MASTER FADER]        │
└──────────────────────────┘
```

### A. **Trim (גם Filter)**

**מה זה:**
- בקרת עוצמת ה-input של כל Deck
- משנה את ה-Gain של הטראק

**למה חשוב:**
- כל טראק הוקלט בעוצמה שונה
- Trim מנרמל את הרמות

**איך להשתמש:**
1. הנגן טראק ראשון
2. בדוק את VU Meter (כָּל הודים ירוק/צהוב)
3. כיוונו Trim עד ש-Peak נמצא ב- -6dB עד -3dB
4. חזור על הטראק השני

**טיפ:** כל Deck צריך להיות באותה רמה בערך כשהם משודכים

### B. **EQ — High, Mid, Low**

**מה זה:**
- שלוש Sliders: HIGH (גבוהים), MID (ביניים), LOW (נמוכים)
- חתימת אודיו של Crossfader

**איך להשתמש:**

**HIGH EQ:**
- משמאל (DOWN) = מסיר צלילים גבוהים (טרבל) — הול מט
- לימין (UP) = מוסיף צלילים גבוהים — קורץ וברקע

**MID EQ:**
- משמאל = הסרת טונים אמצעיים — הוט צר
- לימין = הוספת טונים אמצעיים — מוקד וחימום

**LOW EQ:**
- משמאל = הסרת בס — קל וצח
- לימין = הוספת בס — עמוק וכבד

### תרחיש מעשי:

**מעבר חלק בין שני הטרקים:**
1. יתחילו שני Decks בעוצמה שווה (Trim)
2. כשהבס יתחיל בטראק 2, הורידו את ה-LOW של Deck 1
3. הרימו את ה-LOW של Deck 2 בהדרגה
4. תצפו לעבור חלק בלי "הקטע ירוד"

### C. **Crossfader — מיקס בין Decks**

**מה זה:**
- Slider אופקי באמצע המיקסר
- בקרה על תערובת של Deck 1 ו-Deck 2

**מצבי הנע:**
- קיצוני שמאל = 100% Deck 1
- אמצע = 50% Deck 1 + 50% Deck 2
- קיצוני ימין = 100% Deck 2

**איך להשתמש:**
1. Deck 1 יתחיל בביצוע
2. כשתרצו להציג את Deck 2, הזיזו את ה-Crossfader לימין
3. בדרך כלל בחצי שניה עד שנייה אחת = מעבר חלק

**טיפים:**
- **Slow Crossfader** = מעברים חלקים וזורמים
- **Fast Crossfader** = עברים דרמטיים (scratches)

### D. **Master Volume + Headphone Cue**

**Master Volume:**
- בקרת עוצמה של המוצא לרמקולים
- צריכה להיות בין -6dB ל-0dB בביצוע

**Headphone Cue:**
- בקרת עוצמת האוזניות
- מציעה בקרה עצמאית על עוצמת ה-Cue

## 5. חיבורים — USB, אוזניות, רמקולים

### דיאגרמת חיבורים:

```
┌─────────────────────────────────┐
│    CONNECTIONS (תחתי הבקר)     │
├─────────────────────────────────┤
│                                 │
│  [USB-C] → Computer/Laptop      │
│  [AUX IN] → ערוץ אחר            │
│  [MASTER OUT] → Speakers        │
│  [HEADPHONES] → שקע אוזניות      │
│  [MIC IN] (כמה דגמים)           │
│                                 │
└─────────────────────────────────┘
```

### A. **USB Connection**

**מה זה:**
- חיבור לביתן (Laptop/PC)
- מעביר אודיו + MIDI

**כיצד להגדיר:**
1. חבר USB-C ל-Laptop
2. הפעל Rekordbox
3. בדוק שה-DDJ-FLX4 מזוהה כמכשיר אודיו

**טיפ:** אם אינו מזוהה, הורד Drivers מאתר Pioneer

### B. **Headphones (אוזניות)**

**בחירה טובה:**
- Pioneer HDJ-X5 או HDJ-X10 (מעשית)
- אחרות: Audio-Technica, Sennheiser

**איך להגדיר:**
- חבר ל-Headphones Jack ב-DDJ-FLX4
- בדוק ב-Rekordbox: Output → Headphones

### C. **MASTER Output (לרמקולים)**

**כיצד להתחבר:**
- DDJ-FLX4 → RCA Cables → Powered Speakers

**Powered Speakers טובות:**
- Pioneer DM-40 (לבית)
- Technics SB-200 (פרופסיונלי)
- JBL PRX (מרוכזות)

### D. **AUX Input (חיבור אחר)**

**כיצד:**
- טלפון / דיסק קשיח → AUX
- טלקול בזמן DJ (למשל, שיר ממרחוק)

## 6. Smart Fader Link & Merge FX

### A. **Smart Fader Link**

**מה זה:**
- תכונה מובנה המחברת Crossfader לג'ינגל בקול

**איך זה עובד:**
1. כשתזיזו את ה-Crossfader, ה-Filter מתאים כל צד אוטומטית
2. מונע תתעבורת חדה של בס

**דוגמה:**
- Crossfader שמאל = LOW EQ של Deck 1 מוגבר (בס חזק)
- Crossfader ימין = LOW EQ של Deck 2 מוגבר
- ככל שנעים בימין, בס מ-Deck 1 מתחלש → בס מ-Deck 2 מתחזק

### B. **Merge FX (Effects)**

**מה זה:**
- סוג של אפקט ייחודי ל-FLX4
- משדכים שני Decks בצחקוקים

**תכונות Merge FX:**
- **Echo** — הד של נקודת Crossfader
- **Reverb** — הדהוד על הנקודה
- **Wobble** — ריטמה על קצות הנקודה

**איך להשתמש:**
1. לחץ על Merge FX button
2. בחר בין Echo/Reverb/Wobble
3. הזז את ה-Crossfader = יווצר אפקט

**דוגמה בביצוע:**
- מעבר חלק עם Merge Echo = צליל תלת-מימדי

## 7. Setting Up Your FLX4

### שלב 1: בדוק את כל החיבורים

```
□ USB-C → Laptop
□ Headphones → Headphones Jack
□ MASTER Out → Speakers
□ Power (לחלק מהדגמים)
```

### שלב 2: הפעל את הבקר

- לחץ על הכפתור ההנעה (אם יש)

- סיבוב → יובל שידור בטראק
- יעבוד בעוד בבחירת Vinyl Mode

### שלב 4: בדוק את Pads

- לחץ על כל Pad → צליל מעותק? בחסכון

### שלב 5: בדוק את EQ

- הזז כל EQ → צליל שנשתנה

## הערות המדריך

**טעויות נפוצות:**
1. Trim טרופ נמוך = קול שקט וחמחמן
2. EQ לא מכול = חתך ירוד בין הטרקים
3. Crossfader מהיר מדי = נשמע חד

**ביטחון:**
- אל תעזוב את Jog Wheel בתנועה = יכול לשיתף חום
- אל תיתן זוצק חזק מדי ל-Pads = יכול להיעקם

**טימינג המלמד:**
- שיעור זה צריך להיות עם בקר בפני כולם, אם אפשר
- תן לתלמידים לנגוע בכל כפתור בעצמם
