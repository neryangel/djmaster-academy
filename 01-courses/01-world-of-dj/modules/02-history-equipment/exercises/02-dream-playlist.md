# בניית פלייליסט חלומי

## מטרה

לבנות סט של 10 שירים כאילו אתה DJ — עם חשיבה על BPM, אנרגיה, וזרימה.

## הוראות

### שלב 1: בחר תרחיש (5 דקות)

באיזה מצב היית מנגן?

- **Club Night** (23:00–05:00)
- **חתונה** (21:00–01:00)
- **מסיבת בית** (20:00–00:00)
- **אירוע חוף ביום** (15:00–20:00)
- **סטרימינג אונליין** (אתה בוחר)

### שלב 2: בנה Setlist של 10 שירים (15 דקות)

**כללים:**

- בדיוק 10 שירים
- משך ממוצע: 3–4 דקות לשיר (סה"כ ~35–40 דקות)
- BPM עולה בהדרגה (התחל נמוך, סיים גבוה)
- אנרגיה עולה לאורך הסט

**מלא את הטבלה:**

| # | שיר | אמן | ז'אנר | BPM | אנרגיה (%) |
|---|------|------|--------|-----|------------|
| 1 | | | | ~100 | 30% |
| 2 | | | | ~105 | 40% |
| 3 | | | | ~110 | 50% |
| 4 | | | | ~115 | 60% |
| 5 | | | | ~118 | 65% |
| 6 | | | | ~120 | 70% |
| 7 | | | | ~122 | 75% |
| 8 | | | | ~125 | 85% |
| 9 | | | | ~128 | 90% |
| 10 | | | | ~130 | 100% |

**טיפים:**

- אם אתה יודע את ה-BPM האמיתי — מצוין. אם לא — הערך
- בחר שירים שאתה באמת אוהב, לא רק "שירים מפורסמים"
- חשוב על מעברים: האם שיר 3 עובר טבעי לשיר 4?

### בונוס

צור Playlist ב-Spotify או YouTube עם 10 השירים — ותשמע את הסט מתחילתו לסופו. האם הזרימה עובדת?

## קריטריונים להערכה

| קריטריון | ניקוד |
|----------|-------|
| 10 שירים מלאים עם כל הפרטים | 10 |
| BPM עולה בהדרגה | 5 |
| אנרגיה בעלייה הגיונית | 5 |
| בחירת שירים מגוונת ומעניינת | 5 |
| תרחיש מוגדר והתאמה אליו | 5 |

## הגשה

שלח טבלה מלאה + קישור לפלייליסט (אם יצרת). ציין את התרחיש שבחרת.
