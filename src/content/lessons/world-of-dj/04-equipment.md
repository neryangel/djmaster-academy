# הציוד הבסיסי שצריך היום

כדי להתחיל לנגן כ-DJ, צריך 4 חלקים עיקריים. בואו נכיר כל אחד.

## 4 החלקים החיוניים

```
┌─────────────────────────────┐
│     רמקולים (SPEAKERS)      │
├─────────────────────────────┤
│   בקר / מיקסר (CONTROLLER)  │
├─────────────────────────────┤
│   תוכנה (REKORDBOX)         │
├─────────────────────────────┤
│   מחשב (LAPTOP)             │
└─────────────────────────────┘
```

## 1. מחשב (Laptop)

- **Mac:** M1 ומעלה (מומלץ)
- **Windows:** עם SSD, לפחות 8GB RAM
- **למה חשוב:** מחזיק את כל המוזיקה ומריץ את התוכנה

**דרישות מינימום:** SSD 256GB, RAM 8GB, מסך 13"+

## 2. תוכנה — Rekordbox

Rekordbox היא תוכנת DJ וניהול מוזיקה של Pioneer DJ.

**למה Rekordbox:**
- עובדת בצורה מושלמת עם DDJ-FLX4 (אותה חברה)
- גרסת Free מספיקה להתחלה
- ממשק זהה לציוד מקצועי (CDJ-3000)

**מה היא עושה:**
- ניהול ספריית מוזיקה (BPM, Key, Cue Points)
- ממשק DJ עם שני Decks
- אפקטים ו-Loops
- Beatgrid אוטומטי

## 3. בקר — DDJ-FLX4

**מה זה:** מכשיר פיזי שמתחבר ללפטופ ב-USB ושולט ב-Rekordbox.

**מחיר בישראל:** ₪1,200-1,500

**למה FLX4 ולא אחר:**
- מחיר סביר למתחילים
- שילוב מושלם עם Rekordbox
- Layout מקצועי (לא צעצוע) — דומה לציוד מועדונים
- איכות צליל טובה
- אפשרות שדרוג טבעית לציוד Pioneer מקצועי

## 4. רמקולים

| סוג | דוגמה | מחיר | מתאים ל... |
|------|--------|------|------------|
| מוניטור ביתי | Pioneer DM-40 | ₪400-800 | תרגול בבית |
| רמקול נייד | JBL Eon One Compact | ₪1,500-2,000 | אירועים קטנים |
| מערכת PA | QSC K Series | ₪3,000+ | מועדונים |

**למתחילים:** מוניטורים ביתיים או אפילו אוזניות טובות מספיקים בהתחלה.

## השוואת תוכנות

| תוכנה | מחיר | עובדת עם | דירוג |
|--------|------|----------|-------|
| Rekordbox (Pioneer) | חינם + Pro | DDJ-FLX4 | ⭐⭐⭐ |
| Serato DJ Lite | חינם | בקרים שונים | ⭐⭐ |
| Traktor Lite (NI) | חינם | בקרי NI | ⭐⭐ |
| Virtual DJ | ₪50/חודש | הכל | ⭐ |

> **המלצה שלנו:** Rekordbox היא הבחירה הברורה כי אנחנו עובדים עם DDJ-FLX4. כשתגיעו למועדון — אותה תוכנה, אותו ממשק.

## מה קונים ראשון?

**סדר עדיפויות:**
1. לפטופ (אם אין) — ההשקעה הגדולה
2. DDJ-FLX4 — הלב של ה-Setup
3. אוזניות טובות — חובה לתרגול
4. רמקולים — אפשר להתחיל בלי (עם אוזניות)
