# מבנה שיר — Intro, Build-Up, Drop, Breakdown, Outro

## למה DJ חייב להכיר מבנה שיר?

DJ שלא מכיר מבנה שיר = **נהג שנוהג בלי Waze**. מבנה שיר אומר לך:
- **איפה** להתחיל לערבב (Mix-In Point)
- **מתי** לצפות ל-Drop (כדי לא להפריע)
- **כמה זמן** נשאר עד ה-Outro (כדי להתכונן)
- **איפה** ה-Mix-Out Point לסיום המעבר

---

## המבנה הסטנדרטי — שיר אלקטרוני

```
 ┌──────┬──────┬──────┬──────┬──────┬──────┬──────┐
 │INTRO │BUILD │DROP 1│BREAK │BUILD │DROP 2│OUTRO │
 │      │  UP  │      │DOWN  │  UP  │      │      │
 │16 bar│16 bar│32 bar│16 bar│16 bar│32 bar│16 bar│
 └──────┴──────┴──────┴──────┴──────┴──────┴──────┘
  🟢     🟡     🔴     🟢     🟡     🔴     🟢
  LOW    RISE   PEAK   LOW    RISE   PEAK   LOW   ← Energy
```

### כל חלק בפירוט

#### Intro (16-32 Bars)
- **מה שומעים:** Kick, Hi-Hat, פרקאסיה קלה, אולי אפקט רקע
- **אנרגיה:** 🟢 נמוכה
- **ב-Waveform:** גלים **נמוכים ודקים** — בעיקר כחול (Bass)
- **🎯 Mix-In Point:** זו הנקודה האידיאלית להכניס שיר חדש!

#### Build-Up (8-16 Bars)
- **מה שומעים:** Filter עולה, Snare Roll מתגבר, Riser (צליל שעולה), FX
- **אנרגיה:** 🟡 עולה — מתח!
- **⚠️ אל תעשה מעבר כאן!** הקהל מחכה ל-Drop — אל תפריע!

#### Drop (16-32 Bars)
- **מה שומעים:** Kick חזקה + Bass Line + מלודיה ראשית — **הכל ביחד**
- **אנרגיה:** 🔴 מקסימלית — הרחבה רוקדת!
- **🚫 תן ל-Drop לרוץ!** אל תחליף שיר באמצע Drop — הקהל ייהרס.

#### Breakdown (8-16 Bars)
- **מה שומעים:** ה-Kick נעלמת, מלודיה רכה, ווקאל, אווירה
- **אנרגיה:** 🟢 יורדת — "נשימה" לרחבה
- **DJ Technique:** רגע מצוין להתחיל **להכין** את השיר הבא

#### Drop 2 (16-32 Bars)
- כמו Drop 1, לפעמים עם **שינוי קל** (ווקאל נוסף, סינתיסייזר שונה, אנרגיה גבוהה יותר)

#### Outro (16-32 Bars)
- **מה שומעים:** אלמנטים נעלמים בהדרגה, חוזרת הפרקאסיה
- **אנרגיה:** 🟢 יורדת לנמוך
- **🎯 Mix-Out Point:** כאן מתחילים להוציא את השיר!

## מבנים שונים לפי ז'אנר

### House / Techno (הכי DJ-Friendly)

```
Intro (32 Bars) → Build → Drop (32) → Break (16) → Drop 2 (32) → Outro (32)
```
- Intros ו-Outros **ארוכים** — מעוצבים בדיוק ל-DJs
- **הז'אנר הכי קל למעברים** — הרבה זמן

### Pop / Top 40

```
Intro (4-8) → Verse → Chorus → Verse → Chorus → Bridge → Chorus → Outro (4-8)
```
- Intros **קצרים מאוד** — צריך לעבוד מהר!
- ה-"Drop" = Chorus
- **טיפ:** השתמש ב-Loop להאריך Intro

### Hip-Hop / Trap

```
Intro (4-8) → Verse 1 → Hook → Verse 2 → Hook → Bridge → Hook → Outro
```
- BPM איטי (70-90) עם Hi-Hats מהירים
- ה-Hook = Chorus = נקודת אנרגיה
- **טיפ:** Hard Cut או Backspin בין שירים

### Trance (הכי ארוך)

```
Intro (64) → Build → Drop (32) → Break (32) → Build → Drop 2 (32) → Outro (64)
```
- שירים של **6-10 דקות**
- Breakdowns ארוכים — רגשיים ומלודיים
- המעברים הכי ארוכים ב-DJ (2-3 דקות)

## זיהוי ב-Waveform של Rekordbox

| מה רואים | מה זה | צבע דומיננטי |
|----------|--------|-------------|
| גלים נמוכים ודקים | Intro / Outro | 🔵 כחול |
| גלים שגדלים בהדרגה | Build-up | צבעוני |
| גלים **גבוהים ועבים** | **Drop** | 🔵🟢🔴 מלא |
| ריק / גלים **נמוכים באמצע** | **Breakdown** | 🟢 ירוק |
| גלים שיורדים | Outro | 🔵 כחול |

---

## כלל הזהב

> **Mix-In ב-Intro של שיר B + Mix-Out ב-Outro של שיר A = מעבר טבעי מושלם.**

שני החלקים האלה מעוצבים **בדיוק בשביל מעברים** — הם הנקודות שבהן השירים מתמזגים בצורה הטבעית ביותר.

2. **סמן Hot Cue** על Mix-In Point (🟠) ו-Mix-Out Point (🟣)
3. תרגל מעבר: Outro של A + Intro של B
