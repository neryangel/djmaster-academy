# מבנה שיר — Intro, Drop, Outro

## למה DJ צריך להכיר מבנה שיר?

DJ שלא מכיר מבנה שיר = נהג שלא מכיר את הכביש. מבנה שיר אומר לך:
- **איפה** להתחיל לערבב
- **מתי** לעשות מעבר
- **איפה** לצפות ל-Drop
- **כמה זמן** נשאר לשיר

## המבנה הסטנדרטי (מוזיקה אלקטרונית)

```
INTRO → BUILD-UP → DROP 1 → BREAKDOWN → BUILD-UP 2 → DROP 2 → OUTRO
  16       16        32         16           16          32       16    (תיבות)
```

### Intro (16-32 תיבות)
- **מה שומעים**: Kick, Hi-Hat, פרקאסיה קלה
- **מטרה**: נקודת כניסה לDJ — קל לערבב כאן
- **תכונה**: אנרגיה נמוכה, מעט אלמנטים
- **טיפ לDJ**: זו **נקודת ה-Mix-In** — כאן מתחילים להכניס שיר חדש

### Build-up (8-16 תיבות)
- **מה שומעים**: סינטיסייזרים נכנסים, Filter עולה, Snare Roll
- **מטרה**: ליצור מתח וציפייה
- **תכונה**: אנרגיה עולה בהדרגה
- **טיפ לDJ**: **אל תעשה מעבר כאן** — הקהל חוכה ל-Drop, אל תפריע

### Drop (32 תיבות)
- **מה שומעים**: Kick חזקה, Bass Line, מלודיה ראשית
- **מטרה**: השיא — כאן הרחבה רוקדת
- **תכונה**: אנרגיה מקסימלית
- **טיפ לDJ**: תן לDrop לרוץ! אל תחליף שיר באמצע Drop

### Breakdown (8-16 תיבות)
- **מה שומעים**: ה-Kick נעלמת, מלודיה רכה, אווירה
- **מטרה**: "נשימה" — רגע של מנוחה לפני Drop 2
- **תכונה**: אנרגיה יורדת, מתח נבנה מחדש

### Drop 2 (32 תיבות)
- כמו Drop 1, לפעמים עם שינוי קל (ווקאל נוסף, סינטיסייזר שונה)

### Outro (16-32 תיבות)
- **מה שומעים**: אלמנטים נעלמים בהדרגה, חוזרת הפרקאסיה
- **מטרה**: נקודת יציאה לDJ
- **טיפ לDJ**: זו **נקודת ה-Mix-Out** — כאן מתחילים להוציא את השיר

## מבנים שונים לפי ז'אנר

### House / Techno
```
Intro (32) → Build → Drop (32) → Break (16) → Drop 2 (32) → Outro (32)
```
- Intros ו-Outros ארוכים — מיועדים לDJs
- מעברים חלקים בזמן ארוך

### Pop / Top 40
```
Intro (8) → Verse → Chorus → Verse → Chorus → Bridge → Chorus → Outro (8)
```
- Intros ו-Outros קצרים — צריך לעבוד מהר יותר
- ה-"Drop" הוא הChorus

### Hip-Hop / Trap
```
Intro (8) → Verse 1 → Hook → Verse 2 → Hook → Bridge → Hook → Outro
```
- BPM איטי (70-90) עם Hi-Hats מהירים
- ה-Hook = ה-Chorus = נקודת האנרגיה

## איך מזהים את החלקים?

### בWaveform של Rekordbox
| מה רואים | מה זה |
|----------|--------|
| גלים נמוכים ודקים | Intro / Outro |
| גלים שגדלים בהדרגה | Build-up |
| גלים גבוהים ועבים | Drop |
| ריק / גלים נמוכים באמצע | Breakdown |

### באוזניים
- **Intro**: שומעים רק קצב, בלי Bass חזק
- **Build**: הרגשת "משהו בא" — סאונד שעולה
- **Drop**: BAM — הכל חוזר, הBass חוזר
- **Breakdown**: "אוויר" — רגוע, מלודי
- **Outro**: הצלילים נעלמים, חוזרים לקצב בסיסי

## כלל הזהב: Mix-In ב-Intro, Mix-Out ב-Outro

> **הנקודה הטובה ביותר למעבר היא Intro של שיר B + Outro של שיר A.**
> זה נשמע טבעי כי שני החלקים מעוצבים בדיוק בשביל זה.
