# Beatmatching ידני — תאוריה ודרך

## מה זה Beatmatching ולמה זה חשוב

### ההגדרה

**Beatmatching היא הכושר להנעת שני שירים בדיוק אותו הקצב בו-זמנית.**

### דוגמה ויזואלית

```
Song A (Old):  Kick-Kick-Kick-Kick (בן 128 BPM)
Song B (New):  Kick-Kick-Kick-Kick (בן 125 BPM)

בלי Beatmatching:
Song A: Kick-Kick-Kick [Kick מBPM 128]
Song B: Kick-Kick-[Kick מBPM 125]
                   ↑
                 מחוץ לסנכרון!

עם Beatmatching:
Song A: Kick-Kick-Kick-Kick [128 BPM]
Song B: Kick-Kick-Kick-Kick [128 BPM מתוך דחיפה ל-128]
                   ↑
                 סנכרון מושלם!
```

### למה זה חיוני?

1. **ביצוע חלק** — כשBeat מסונכרן, מעבר נוער
2. **Mixing Effects** — EQ, Loops עובדים רק בסנכרון
3. **Professionalism** — מראה שאתה DJ טוב
4. **אדם רוקדים** — קהל יודע כשהביטים כפופים!

## Pitch Fader — שליטה במהירות

### מה זה?

Slider בכל Deck שמשנה את מהירות הטראק.

### איך זה עובד

```
Pitch Fader Position → BPM Change

-8%    →  128 BPM × 0.92 = 117.7 BPM
-5%    →  128 BPM × 0.95 = 121.6 BPM
-3%    →  128 BPM × 0.97 = 124.2 BPM
0%     →  128 BPM (Original)
+3%    →  128 BPM × 1.03 = 131.8 BPM
+5%    →  128 BPM × 1.05 = 134.4 BPM
+8%    →  128 BPM × 1.08 = 138.2 BPM
```

### DDJ-FLX4 Pitch Fader

```
על בקר:
Slow (DOWN) ← Pitch Fader → Fast (UP)
-100%                      0%                      +100%
(חצי מהמהירות)           (מהירות מקורית)      (כפול מהמהירות)
```

### טיפים Pro

- **Pitch Range** — בדרך כלל ±8% נוח ל-DJ
- **תנועה קטנה** = דיוק טוב
- **Pitch Slope** — "Medium" בסדר

## האזנה עם Headphones — Cue & Master Mix

### מבנה האוזניות

```
Headphones Volume (בקרה)
   ↓
Cue/Master Mix Button (בחירה מה לשמוע)
   ↓
טיפה בDeck בקרה (כמה מכל)
```

### מה זה Cue?

**Cue = כמה לשמוע של הטראק החדש (Song B) באוזניות.**

```
Cue Slider לשמאל (DOWN) → 100% Song B
                        → 0% Song A

Cue Slider לאמצע → 50% Song B + 50% Song A

Cue Slider לימין (UP) → 0% Song B
                      → 100% Song A
```

### תהליך Beatmatching עם Cue

```
1. עדיין נוגן Song A עם הרמקול
2. טען Song B ל-Deck 2
3. הנחת Song B בHeadphones (Cue)
4. שנה Pitch של Deck 2 = Deck 1 BPM
5. כשה-Beats מסונכרנים → עבור לSong B
```

## זיהוי Beat הראשון

### מה זה?

**הKick (בס) הראשון של הטראק.**

### איך לזהות

1. **תנגן את Intro של טראק**
2. **בדוק את Waveform (גרף)**
   - תבט לתחילה
   - Kick הראשון = קו בולט בגרף
3. **שמע את זה**
   - Kick הראשון נוער בתחילה
4. **הגדר Hot Cue בנקודה זו**

### דוגמה ביזואלית

```
Waveform:
│
│ ╱╲ ╱╲ ╱╲      ← צלילים
│╱  ╲╱  ╲╱
└─────────────
0:00 ← Beat הראשון (Kick)

Grid:
│▌  ▌  ▌  ▌  ← Grid Lines (צריך להיות בדיוק על Kicks)
└─────────────
```

## Nudging — דחיפה עדינה

### מה זה?

דחיפה עדינה של Jog Wheel (בלי לשנות BPM) כדי להתקדם או לאחור קמן.

### שתי דרכים

#### A. Jog Wheel — Platter Mode

גרור את האצבע:
- **למעלה** = קדימה מעט
- **למטה** = אחורה מעט

#### B. Pitch Bend

לחץ ותחזוק על כפתור — זה יוצר "Bend" זמני.

### כמה זה משהו חשוב?

בעד 120 BPM:
- Bar אחת = 2 שניות
- Beat אחת = 0.5 שניות
- Nudge של 0.1 שניות = Nudge קטן אבל שמורגש

### דוגמה

```
Song A: Kick-Kick-Kick-Kick [ב-Beat 1,2,3,4]
Song B: Kick-Kick-[Kick]-Kick [מעט מאחור]
                   ↓
        Nudge Song B קדימה

Song A: Kick-Kick-Kick-Kick [עדיין ב-Beat 1,2,3,4]
Song B: Kick-Kick-Kick-Kick [עכשיו תואם!]
```

## תיקון Drift בזמן אמת

### מה זה Drift?

**סטיה קטנה בין שני הטרקים עקב Tempo Variations בטואל.**

### דוגמה

```
זמן 0:00:  שני הטרקים מסונכרנים (100%)
זמן 1:00:  Song B מעט קדימה (Drift!)
זמן 2:00:  Song B עוד יותר קדימה (20 שניות לפני Song A!)
```

### איך לתקן

1. **הקשב ל-Headphones (Cue)**
2. **בדוק: האם שני Kicks מסונכרנים?**
3. **אם Drift:**
   - Nudge Song B קצת
   - או שנה Pitch ב-0.1-0.2%

### טיפים Pro

- **Nudge** = מהיר (לשניות)
- **Pitch** = איטי (לדקות)
- **בדוק כל 30 שניות** לתיקון

## Beatmatching בפועל — שלב אחר שלב

### הכנה

1. **Sync כבוי** — לא רוצים עזרה אוטומטית
2. **Song A נוגן** — לפחות 30 שניות
3. **Song B טעון** — מוכן ב-Deck 2

### שלבים

#### 1. טען Song B
```
Deck 2 → טען Song B (בשקט, בHeadphones)
```

#### 2. בדוק BPM Rough
```
BPM Display:
Song A: 128 BPM
Song B: 126 BPM
Difference: 2 BPM (קטן!)
```

#### 3. שנה Pitch של Song B
```
Pitch Fader של Deck 2:
תנוע מ-0% → +1.5%

126 × 1.015 = 127.9 ≈ 128 BPM ✓
```

#### 4. האזן בHeadphones
```
Headphones: Cue of Song B
בדוק: האם Kicks של שני הטרקים תואמים?
אם כן → עבור לשלב 5
אם לא → תיקון Pitch / Nudge
```

#### 5. עבור ל-Master Mix
```
Headphones מ-Cue Only → Cue/Master Mix
עכשיו שומע שניהם
בדוק: שניים בסנכרון?
```

#### 6. בצע Crossfader
```
Crossfader מ-Deck A → Center → Deck B
לאט, ~2 שניות

בדוק: מעבר חלק? ללא "Drops"?
```

#### 7. השבתת Song A
```
כשCrossfader = 100% Song B:
- כבה את Song A אם רוצה
- או תן לו לרדת לאט
```

## בעיות נפוצות

### בעיה 1: הטראק נשמע מתין לאט

**סיבה:** Pitch חיובי מדי
**פתרון:** הורד את Pitch ב-0.5-1%

### בעיה 2: הBeat מתפרק כל כמה שניות

**סיבה:** Drift
**פתרון:** Nudge כל 30 שניות

### בעיה 3: אני לא שומע את הBeat בדיוק

**סיבה:** Headphones Volume נמוכה או שום Cue
**פתרון:** הרם Volume, חזור לCue

### בעיה 4: הBeat מהנבחר כבוי מדי

**סיבה:** Pitch בקיצוניות (אבל +20%)
**פתרון:** צמצם Pitch Range, בחר BPM דומה

## תרגול שיטתי

### שלב ראשון (5-10 דקות)
- שני שירים בעד 120-128 BPM
- בצע Beatmatching 10 פעמים
- בדוק שכל פעם תוקן ב< 30 שניות

### שלב שני (10-15 דקות)
- שני שירים בBPM שונה (5-10 BPM סטיה)
- בצע Beatmatching 10 פעמים
- בדוק שתוקנו ב< 45 שניות

### שלב שלישי (15-20 דקות)
- בנה Set של 3-4 שירים
- בצע Beatmatching ללא הפסקה
- בדוק ש-Transitions חלקות
