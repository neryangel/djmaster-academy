# סקירת DDJ-FLX4 — מבנה וחלקים

ה-DDJ-FLX4 הוא בקר DJ ממדורג (2 Deck) עם מיקסר מובנה. זה הלב של ה-Setup שלך.

## המבנה הכללי

```
┌────────────────────────────────────────────────┐
│         PIONEER DDJ-FLX4 — TOP VIEW            │
├────────────────────────────────────────────────┤
│                                                │
│  [DECK 1]        [MIXER]        [DECK 2]     │
│  ┌─────────┐   ┌──────────┐   ┌─────────┐   │
│  │ Jog Whl │   │ Faders   │   │ Jog Whl │   │
│  │  (L)    │   │   EQ     │   │  (R)    │   │
│  │         │   │  Trim    │   │         │   │
│  └─────────┘   └──────────┘   └─────────┘   │
│                                                │
│  ┌─────────┐   ┌──────────┐   ┌─────────┐   │
│  │  Pads   │   │ Buttons  │   │  Pads   │   │
│  │  8x2    │   │  Volume  │   │  8x2    │   │
│  │         │   │  Cue     │   │         │   │
│  └─────────┘   └──────────┘   └─────────┘   │
│                                                │
│  [USB] [AUX] [HEADPHONES] [MASTER]          │
│                                                │
└────────────────────────────────────────────────┘
```

## 4 אזורים עיקריים

1. **Deck 1 (שמאל)** — בקרה על הטראק הראשון
2. **Deck 2 (ימין)** — בקרה על הטראק השני
3. **Mixer Section (אמצע)** — ערבוב בין הטרקים
4. **Buttons & Controls (תחתון)** — בקרים כללים

## Jog Wheels — שימוש ופונקציות

### Platter Mode — מצב ויניל

גרור את האצבע על הגלגל:
- **למעלה** = קדימה
- **למטה** = אחורה
- כמו טרנטייבל ישן — בקרה טבעית

### CDJ Mode — מצב כתום

גרור למעלה או למטה:
- בקרה ליניארית
- יותר מדויקת ל-scratching

### Pitch Bend

גרור במהירות = bend זמני (משחרר = חוזר לנורמלי)

**טיפ Pro:** Vinyl Mode הוא ברירת המחדל — התחל בזה!

## Performance Pads — 8 כפתורים בכל Deck

לכל Pad יש 4 מודים:

### 1. Hot Cue Mode — סימנים מהירים

סימן נקודה בשיר — לחיצה קופצת לשם.

**דוגמה:**
- Pad 1: Intro (0:00)
- Pad 2: Verse (0:30)
- Pad 3: Chorus (1:00)

### 2. Loop Mode — לולאות חוזרות

תקוע חלק מהטראק בחזרה אינסופית.

**דוגמה:** Loop על ה-Chorus = חוזרים פי 3

### 3. Slicer Mode — פרוסות קצובות

חלוקת 8 beats ל-8 חתיכות — נגן בסדר או אקראי.

### 4. Sampler Mode — דגימות

הקלט קול וניגן מחדש בלחיצה.

## Mixer Section

```
┌──────────────────────────┐
│    MIXER SECTION         │
├──────────────────────────┤
│  [TRIM] [TRIM]           │
│   L        R             │
│                          │
│  [EQ HIGH] [EQ HIGH]     │
│  [EQ MID]  [EQ MID]      │
│  [EQ LOW]  [EQ LOW]      │
│                          │
│    [CROSSFADER]          │
│    [MASTER FADER]        │
└──────────────────────────┘
```

### Trim — בקרת עוצמה

כל טראק הוקלט בעוצמה שונה — Trim מנרמל את הרמות.

### EQ — High, Mid, Low

**HIGH:** צלילים גבוהים (טרבל)
**MID:** טונים בינוניים
**LOW:** בסים

### Crossfader — מיקס בין Decks

- קיצוני שמאל = 100% Deck 1
- אמצע = 50% Deck 1 + 50% Deck 2
- קיצוני ימין = 100% Deck 2

## חיבורים — USB, אוזניות, רמקולים

```
┌─────────────────────────────┐
│  CONNECTIONS (תחתי הבקר)   │
├─────────────────────────────┤
│                             │
│  [USB-C] → Computer/Laptop  │
│  [AUX IN] → ערוץ אחר        │
│  [MASTER OUT] → Speakers    │
│  [HEADPHONES] → אוזניות      │
│                             │
└─────────────────────────────┘
```

### USB Connection
חיבור ללפטופ — מעביר אודיו + MIDI

### Headphones
שמיעה מוקדמת של הטראק הבא (Cue)

### MASTER Output
יציאה לרמקולים

### AUX Input
קישור מקור אחר (טלפון, דיסק)

## Smart Fader Link & Merge FX

### Smart Fader Link
תכונה שמחברת Crossfader ל-Filter אוטומטית — יוצרת מעברים חלקים יותר.

### Merge FX
אפקטים ייחודיים:
- **Echo** — הד בנקודת Crossfader
- **Reverb** — הדהוד
- **Wobble** — ריטמה

## הגדרה ראשונית

```
□ USB-C → Laptop
□ Headphones → Headphones Jack
□ MASTER Out → Speakers
□ Power (אם יש)
```

כעת אתה מוכן להשתמש בכל חלקי הבקר!
