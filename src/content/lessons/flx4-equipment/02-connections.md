# חיבורים וקישוריות

## דיאגרמת חיבורים

```
┌─────────────────────────────────┐
│  CONNECTIONS (תחתי הבקר)       │
├─────────────────────────────────┤
│                                 │
│  [USB-C] → Computer/Laptop      │
│  [AUX IN] → ערוץ אחר            │
│  [MASTER OUT] → Speakers        │
│  [HEADPHONES] → אוזניות          │
│  [MIC IN] (בחלק מהדגמים)        │
│                                 │
└─────────────────────────────────┘
```

## 1. USB Connection — חיבור ללפטופ

### מה זה עושה
חיבור USB-C לביתן עם הלפטופ/PC.
- מעביר אודיו מהתוכנה לבקר
- מעביר MIDI (בקרים מהבקר לתוכנה)

### כיצד להגדיר

1. **חבר USB-C ל-Laptop**
   - יציאת USB-C בתחתית הבקר
   - יציאת USB-C או USB-A בלפטופ (אם צריך, גרום מתאם)

2. **הפעל Rekordbox**
   - התוכנה צריכה לזהות את הבקר תוך שניות
   - תראה הודעה: "DDJ-FLX4 זוהה"

3. **בדוק את ההגדרות**
   - Preferences → Audio
   - Primary Output: DDJ-FLX4 (MASTER)
   - Headphones Output: DDJ-FLX4 (HEADPHONES)

### טיפים
- **אם לא מזוהה:** הורד Drivers מ-pioneer-dj.com
- **החבר קודם, אחרי זה הפעל Rekordbox**
- **בדוק שהכבל בתוקף** — כבלים USB רעים הם הבעיה השכיחה ביותר

## 2. Headphones (אוזניות)

### חיבור פיזי

1. **קח את אוזניות ה-DJ שלך**
   - צריך להיות jack 3.5mm סטנדרטי (או jack גדול 6.35mm)

2. **חבר לתחתית הבקר**
   - יציאת "HEADPHONES" בתחתית הבקר
   - זה בדרך כלל יציאה עם סמל אוזניות

### הגדרה בתוכנה

**Rekordbox:**
1. Preferences → Audio
2. Headphones Output: DDJ-FLX4 (HEADPHONES)
3. Headphones Volume: כוונן בחזה של הבקר

### בחירת אוזניות טובות

| מודל | מחיר | דירוג |
|------|------|-------|
| Pioneer HDJ-X5 | ₪500-800 | ⭐⭐⭐ |
| Pioneer HDJ-X10 | ₪800-1,200 | ⭐⭐⭐⭐ |
| Audio-Technica ATH-M50X | ₪400-600 | ⭐⭐⭐ |
| Sennheiser HD25 | ₪600-900 | ⭐⭐⭐⭐ |

**למתחילים:** אוזניות טובות עם עמידות וצליל שטוח מומלצות.

## 3. MASTER Output — חיבור לרמקולים

### כבלים שצריך

**RCA Cables** (אדום + לבן):
```
DDJ-FLX4 MASTER OUT (RCA) → Speakers (RCA IN)
```

### סוגי רמקולים

#### Powered Speakers (מומלץ להתחלה)
- עם מגבר מובנה
- לא צריך מגבר נוסף
- דוגמאות: Pioneer DM-40, JBL LSR305

**מחיר:** ₪400-2,000

#### Passive Speakers
- צריך מגבר חיצוני
- טיב צליל טוב יותר בעלות גבוהה
- בחרו למועדונים/סטודיו

### שלבים להתחבר

1. **תחת הבקר, חפש "MASTER OUT"**
   - שתי יציאות RCA (אדום + לבן)

2. **חבר RCA cables**
   - אדום לאדום (Right)
   - לבן ללבן (Left)

3. **חבר לרמקולים**
   - כניסות RCA בגב הרמקולים

4. **בדוק את הרמקולים**
   - בדרך כלל יש כפתור Volume בכל רמקול
   - התחל ברמקול בנמוך (כ-30%)

### בדיקה שהקול עובד

1. הפעל Rekordbox
2. טען שיר
3. הגבר את Master Volume בבקר (למעלה לאט)
4. אתה צריך לשמוע קול מהרמקולים

## 4. AUX Input — חיבור מקור אחר

### מה זה משמש

חיבור טלפון, דיסק קשיח, או ציוד אחר ישירות לבקר.

**דוגמה:** שחקן טלקול שמצלצל בזמן DJ ואתה רוצה להשמיע אותו לקהל.

### חיבור

```
Telephone/Device (3.5mm) → AUX IN (יציאה בתחתית הבקר)
```

### שימוש

1. חבר את המקור ל-AUX
2. ב-Rekordbox: Preferences → Input
3. בחר AUX כ-Input
4. כוונן את Volume של AUX בבקר

## בדיקת כל הקישורים

```
□ USB-C → Laptop (Rekordbox זוהה?)
□ Headphones → Headphones Jack (שומע בהם?)
□ MASTER OUT → Speakers (קול יוצא?)
□ AUX IN (אם צריך) → מקור אחר
```

## בעיות נפוצות וחיסול בעיות

### בעיה: "אין קול מהרמקולים"

**בדוק:**
1. האם הבקר מחובר ב-USB?
2. האם Rekordbox רואה את הבקר?
3. האם Master Volume במקום לא?
4. האם הרמקולים מופעלים?

### בעיה: "האוזניות שקטות מדי"

**בדוק:**
1. כוונן Headphones Volume בבקר (למעלה)
2. בדוק ב-Rekordbox: Preferences → Output

### בעיה: "אין MIDI מהבקר"

**בדוק:**
1. הוא מחובר ב-USB?
2. Rekordbox זוהה אותו?
3. הורד Drivers אם צריך
