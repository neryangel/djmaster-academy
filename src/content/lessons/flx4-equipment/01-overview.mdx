# סקירת DDJ-FLX4 — מבנה, חלקים, ו-Signal Flow

ה-DDJ-FLX4 הוא בקר DJ דו-ערוצי (2-Channel) עם מיקסר מובנה ו-Audio Interface מובנה. **זה הלב של ה-Setup שלך** — הכל מתחיל ונגמר כאן.

---

## מפרט טכני מלא

| מאפיין | ערך | הערה |
|---------|------|-------|
| **מידות** | 482 × 272.8 × 59.2 mm | רוחב כמו מגש סטנדרטי |
| **משקל** | 2.1 kg | קל מאוד — נכנס בתיק |
| **Jog Wheels** | 111.6 mm קוטר | קטנים אבל מדויקים |
| **Performance Pads** | 8 × 2 (16 סה"כ) | רגישים ללחיצה |
| **Audio Resolution** | 16-bit / 24-bit | 24-bit = איכות סטודיו |
| **Sample Rate** | 44.1 kHz / 48 kHz | 48kHz = תדר מקצועי |
| **Frequency Response** | 20 Hz – 20,000 Hz | טווח השמיעה האנושי המלא |
| **S/N Ratio** | 103 dB | יחס רעש מצוין |
| **THD** | < 0.005% (USB) | עיוות כמעט אפס |
| **חיבורים** | USB-C ×2, RCA, 3.5mm, ¼" MIC | כל מה שצריך |
| **תאימות** | Rekordbox, Serato, Mac, PC, iOS, Android | Multi-Platform |

## Signal Flow — איך הצליל זורם

הבנת Signal Flow היא **קריטית לDJ**. כשאתה יודע את הדרך של הצליל, אתה יודע **איפה הבעיה** כשמשהו לא עובד.

```
  קובץ מוזיקה (MP3/WAV על המחשב)
        │
        ▼
  ┌─────────────────┐
  │   REKORDBOX      │  ← תוכנה: ניתוח, Waveform, Effects
  │   (Software)     │
  └────────┬────────┘
           │ USB-C (Digital Audio + MIDI)
           ▼
  ┌─────────────────────────────────────────┐
  │           DDJ-FLX4 (Hardware)           │
  │                                         │
  │  ┌─────────┐         ┌─────────┐       │
  │  │ DECK A  │         │ DECK B  │       │
  │  │(שמאל)   │         │(ימין)   │       │
  │  └────┬────┘         └────┬────┘       │
  │       │                    │            │
  │       ▼                    ▼            │
  │  ┌──────────────────────────────┐       │
  │  │        MIXER SECTION         │       │
  │  │  Trim → EQ → Filter → Fader │       │
  │  │         ↓     ↓              │       │
  │  │    Crossfader → Master       │       │
  │  └───────────┬──────────────────┘       │
  │              │                          │
  │     ┌────────┼────────┐                 │
  │     │        │        │                 │
  │     ▼        ▼        ▼                 │
  │  MASTER   PHONES    MIC                 │
  │  OUT      OUT       IN                  │
  └─────────────────────────────────────────┘
        │        │
        ▼        ▼
    רמקולים   אוזניות
   (קהל)     (DJ בלבד)
```

**נקודות מפתח:**
1. הצליל מתחיל כקובץ דיגיטלי על המחשב
2. עובר ב-USB-C לבקר (MIDI חוזר מהבקר לתוכנה)
3. בבקר — עובר דרך Trim → EQ → Filter → Fader → Crossfader → Master
4. יוצא כאות אנלוגי מ-RCA לרמקולים **ו** מ-3.5mm לאוזניות

## 5 אזורים — סיור מודרך

### אזור 1: Deck Section (×2)

כל Deck שולט על שיר אחד. שני ה-Decks **זהים במבנה** — שמאל (Deck A) וימין (Deck B).

**מה יש בכל Deck:**

| רכיב | מה עושה | מה קורה כשלוחצים |
|------|---------|-----------------|
| **Play/Pause** | מנגן/עוצר שיר | הכפתור הגדול — הרגע הראשון |
| **Cue** | חוזר ל-Cue Point | בלחיצה = קפיצה, בהחזקה = השמעה זמנית |
| **Jog Wheel** | Scratch, Nudge, Search | Vinyl Mode: נגיעה = עצירה. Edge = Pitch Bend |
| **Pitch Fader** | שינוי BPM | למעלה = מאיץ, למטה = מאט |
| **Performance Pads** | 8 כפתורים נשלטים | Hot Cue, Loop, Beat Jump, Sampler + עוד |

> **מה בדיוק קורה כשלוחצים Play?** Rekordbox שולחת את האות הדיגיטלי של השיר דרך USB-C → הבקר ממיר אותו לאנלוגי (DAC) → יוצא לרמקולים.

### אזור 2: Mixer Section (אמצע)

המיקסר שולט על **מה הקהל שומע** ובאיזה איזון.

| רכיב | תפקיד | משפיע על |
|------|--------|---------|
| **Trim (×2)** | עוצמת כניסה לכל ערוץ | Input Level |
| **EQ High (×2)** | תדרים גבוהים (2kHz-20kHz) | Hi-Hats, Cymbals |
| **EQ Mid (×2)** | תדרים בינוניים (250Hz-2kHz) | Vocals, Synths |
| **EQ Low (×2)** | תדרים נמוכים (20Hz-250Hz) | Kick, Bass |
| **Filter (×2)** | Low Pass / High Pass | סינון תדרים |
| **Channel Fader (×2)** | עוצמת ערוץ | Volume |
| **Crossfader** | איזון A ↔ B | Balance |
| **Master** | עוצמה כללית | Output Level |

### אזור 3: Performance Pads (×2)

8 Pads בכל Deck — 16 סה"כ. כל Pad רגיש ללחיצה ודולק בצבע.

**6 מצבים זמינים:**
1. **Hot Cue** — קפיצה לנקודות סימון (הכי חשוב!)
2. **Beat Loop** — לולאות חוזרות
3. **Beat Jump** — קפיצה קדימה/אחורה
4. **Sampler** — הפעלת צלילים
5. **Key Shift** — שינוי מפתח בזמן אמת
6. **Pad FX** — אפקטים בלחיצת Pad

### אזור 4: Transport & Mode Buttons

| כפתור | תפקיד |
|--------|--------|
| **SHIFT** | גישה לפונקציות משניות (כמו Alt בממשק) |
| **VINYL** | מעבר בין Vinyl Mode ל-Normal Mode |
| **SYNC** | סנכרון BPM אוטומטי |
| **Smart Fader** | הפעלת Smart Fader Link |
| **MERGE FX** | מעבר דרמטי מובנה |
| **Smart CFX** | אפקטים חכמים |

### אזור 5: חיבורים (תחתית ואחורה)

| חיבור | סוג | מה מתחבר | הערה |
|--------|------|----------|------|
| **USB-C #1** | USB-C | מחשב | Audio + MIDI + חשמל |
| **USB-C #2** | USB-C | מקור חשמל נוסף | לטעינה בלבד |
| **MASTER OUT** | RCA (L+R) | רמקולים | אדום=ימין, לבן=שמאל |
| **PHONES** | 3.5mm | אוזניות | Cue / Master Mix |
| **MIC** | ¼" (6.35mm) | מיקרופון | עם Level Control |

## השוואה לציוד מועדוני — CDJ-3000 + DJM-900NXS2

| קריטריון | DDJ-FLX4 | CDJ-3000 + DJM-900 |
|----------|----------|---------------------|
| **מחיר** | ~₪1,300 | ~₪35,000+ |
| **Jog Wheels** | 111.6mm | 206mm |
| **מסכים** | אין (על הלפטופ) | 9" מגע צבעוני |
| **Audio** | 24-bit/48kHz | 24-bit/96kHz |
| **USB** | צריך מחשב | עובד עם USB בלבד |
| **EQ** | 3-Band | 4-Band (Isolator) |

> **הנקודה החשובה:** ה-Layout של ה-FLX4 **מבוסס על CDJ-3000**. מה שתלמד על FLX4 — תדע להפעיל גם CDJ. **זה בדיוק למה בחרנו בו.**

## סיור מודרך ראשון — 10 דקות על הבקר

### שלב 1: נגע בכל דבר (3 דק')

בלי להפעיל שום דבר — פשוט **נגע ותרגיש**:
- סובב את ה-Jog Wheels — הרגש את ההתנגדות
- הזז את ה-Pitch Fader — למעלה ולמטה
- הזז את Channel Faders ו-Crossfader
- סובב כפתורי EQ ו-Filter
- לחץ על Pads — הרגש את הרגישות

### שלב 2: חבר והפעל (2 דק')

1. USB-C → מחשב
2. אוזניות → Phones Jack
3. פתח Rekordbox
4. ודא "DDJ-FLX4 Connected"

### שלב 3: טען שיר ראשון (2 דק')

1. גרור שיר ל-Deck A
2. לחץ Play
3. שמע באוזניות
4. שחק עם Volume (Channel Fader)

### שלב 4: שחק עם EQ (3 דק')

1. בזמן שהשיר מנגן — סובב EQ Low לאפס → הבסים נעלמים
2. החזר Low → סובב EQ High לאפס → הגבוהים נעלמים
3. שים לב: **Mid הכי רגיש** — שינוי קטן = שינוי גדול

> **הישג:** אחרי 10 דקות, אתה מכיר כל חלק פיזי בבקר ויודע מה כל כפתור עושה. **זה הבסיס לכל מה שילמד.**
