# Performance Pads — Hot Cue, Loop, Beat Jump, Sampler

## מה הם Performance Pads?

ה-DDJ-FLX4 כולל **8 Pads בכל Deck** — 4 למעלה ו-4 למטה, סה"כ 16 Pads. הם **רגישים ללחיצה** ודולקים בצבעים שונים לפי הפונקציה.

```
┌─────────────────────────┐
│  Performance Pads       │
│  ┌───┬───┬───┬───┐      │
│  │ 1 │ 2 │ 3 │ 4 │ ← שורה עליונה
│  ├───┼───┼───┼───┤      │
│  │ 5 │ 6 │ 7 │ 8 │ ← שורה תחתונה
│  └───┴───┴───┴───┘      │
│                         │
│  [HOT CUE] [LOOP]       │ ← כפתורי מצב
│  [BEAT JUMP] [SAMPLER]   │
└─────────────────────────┘
```

## מצב 1: Hot Cue — הכי חשוב!

### מה זה?

Hot Cue מאפשר **לסמן נקודות ספציפיות** בשיר. לחיצה על Pad = קפיצה **מיידית** לאותה נקודה, בלי עיכוב.

### אסטרטגיית Hot Cue מקצועית — 8 נקודות

DJ מקצועי מסמן **כל שיר חדש** עם Hot Cues לפני ההופעה:

| Pad | נקודה | צבע | מתי משתמשים |
|-----|--------|------|-----------|
| **1** | Intro Start | 🟢 ירוק | תחילת השיר — נקודה ראשונה |
| **2** | Verse / Vocal | 🔵 כחול | תחילת הווקאל |
| **3** | Breakdown | 🟡 צהוב | תחילת Breakdown (מתח) |
| **4** | Drop | 🔴 אדום | ה-Drop! הרגע החזק ביותר |
| **5** | Mix-In Point | 🟠 כתום | נקודה אידיאלית להתחיל מעבר |
| **6** | Mix-Out Point | 🟣 סגול | נקודת יציאה אידיאלית |
| **7** | Build-Up | ⚪ לבן | תחילת Build-up |
| **8** | Outro/Special | 🟤 חום | Outro או אלמנט מיוחד |

### איך להגדיר Hot Cue

1. נגן שיר ב-Rekordbox
2. כשמגיע **לנקודה הרצויה** — לחץ על Pad ריק
3. ה-Pad ידלק בצבע — ה-Cue **נשמר לצמיתות** (גם אחרי סגירת Rekordbox)
4. **למחוק Cue:** SHIFT + לחיצה על ה-Pad

> **כלל ברזל:** הגדר Hot Cues **לפני** ההופעה, לא תוך כדי. הכנה מסודרת = ביצוע חלק ומקצועי.

## מצב 2: Beat Loop — לולאות

### מה זה?

Beat Loop יוצר **לולאה שחוזרת** על עצמה באורך מוגדר. ברגע שמפעילים — החלק הזה של השיר חוזר שוב ושוב עד ששחרר.

### אורכי Loop ושימושים

| Pad | אורך | Beats | שימוש מקצועי |
|-----|-------|-------|-------------|
| 1 | **1/4 Beat** | 0.25 | אפקט "מקלע" — מאוד דרמטי |
| 2 | **1/2 Beat** | 0.5 | Build-up מהיר, ריצפי |
| 3 | **1 Beat** | 1 | Loop בסיסי, אפקט חזרה |
| 4 | **2 Beats** | 2 | Loop קצר אבל מוזיקלי |
| 5 | **4 Beats** | 4 | **תיבה שלמה** — הכי שימושי |
| 6 | **8 Beats** | 8 | שני תיבות — למעברים |
| 7 | **16 Beats** | 16 | חצי משפט — מעבר ארוך |
| 8 | **32 Beats** | 32 | משפט שלם — מעבר מלא |

### טכניקה: Loop + Fade (מעבר קלאסי)

```
שלב 1: שיר A מנגן, שיר B מוכן
שלב 2: הפעל Loop 8 Beats על שיר A ← A חוזר על עצמו
שלב 3: הכנס את שיר B בהדרגه (Fader למעלה)
שלב 4: הנמך את Fader של A בהדרגה
שלב 5: שחרר Loop ← A ממשיך ונעלם, B לוקח שליטה
```

### טכניקה: Loop Stutter (Build-up)

1. הפעל Loop **4 Beats**
2. אחרי 2 חזרות — קצר ל-**2 Beats**
3. אחרי 2 חזרות — קצר ל-**1 Beat**
4. אחרי 2 חזרות — קצר ל-**1/2 Beat**
5. **שחרר!** → הDrop פוגע → אנרגיה מטורפת

## מצב 3: Beat Jump

**Beat Jump מקפיץ קדימה או אחורה** בשיר בלי לעצור את הקצב ובלי לשנות את הBPM.

### שימושים מקצועיים

| שימוש | למה | איך |
|--------|-----|------|
| **דלג על Intro ארוך** | חוסך זמן | Jump +32 beats |
| **חזור לנקודה שפספסת** | תיקון טעות | Jump -16 beats |
| **אפקט "Cut"** | הפתעה | Jump +8 ← שינוי פתאומי |
| **סנכרון Phrases** | יישור משפטים | Jump ±4 beats |

## מצב 4: Sampler

**Sampler מאפשר לטעון ולהפעיל צלילים קצרים** בלחיצה:

| שימוש | דוגמה |
|--------|--------|
| **אפקטים** | אלקטרוני, סירנה, לייזר |
| **ווקאלים** | "Let's go!", "One more time!" |
| **ג'ינגלים** | שם ה-DJ, לוגו קולי |
| **לולאות** | Loop של Beat שמתנגן על כל שיר |

> **טיפ Pro:** צור ספריית Samples אישית עם 8-16 צלילים ייחודיים. זה מה שנותן לך **חתימה קולית** כ-DJ.

## מצב 5: Key Shift

Key Shift **משנה את המפתח המוזיקלי** של השיר בזמן אמת:
- כל Pad = שינוי של **חצי טון** (semitone)
- **למעלה** = Pitch גבוה יותר
- **למטה** = Pitch נמוך יותר

**מתי להשתמש:**
- שני שירים עם Key שלא תואם — שנה את אחד מהם
- **Energy Boost** — העלה חצי טון = הרגשת "הרמה"

---

## מצב 6: Roll (לולאת שחרור)

**Roll = Loop שלא מפספס!** כשמשחררים את ה-Pad, **השיר ממשיך מהנקודה שהיה מגיע אליה** אילולא ה-Loop.

```
ללא Roll:
A-B-C-D-[LOOP: C-C-C-C]-C-D-E-F ← חוזר מ-C

עם Roll:
A-B-C-D-[ROLL: D-D-D-D]-E-F-G-H ← ממשיך מ-E!
```

**למה זה גאוני?** אפשר לעשות Build-up מטורף עם Roll ו**לא לפספס את ה-Drop** — הוא ימשיך בדיוק בזמן.

## תרגולים מובנים

### תרגיל 1: Hot Cue Setup (10 דק')

1. **טען 3 שירים** ל-Rekordbox
2. **סמן 4 Hot Cues** בכל שיר: Intro, Drop, Breakdown, Outro
3. **תרגל קפיצה** — עבור בין Cues ברגע

### תרגיל 2: Loop Build-Up (10 דק')

1. נגן שיר → **הפעל Loop 4 beats**
2. קצר: 4 → 2 → 1 → ½ → ¼
3. **שחרר ב-Drop!**
4. חזור 5 פעמים עד שהתזמון מושלם

### תרגיל 3: Beat Jump Navigation (5 דק')

1. נגן שיר → **דלג +32 beats** (משפט שלם)
2. **חזור -16 beats**
3. **דלג שוב +8** — תגיע לDrop?
4. למד "לנווט" בשיר עם Jump

### תרגיל 4: Roll + Drop (10 דק')

1. נגן שיר עם Build-Up ברור
2. **הפעל Roll** בתחילת ה-Build-Up
3. קצר את ה-Roll בהדרגה
4. **שחרר בדיוק על ה-Drop** — הכל מתפוצץ!
