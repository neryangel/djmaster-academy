# ניהול ספריית מוזיקה

## למה ארגון חשוב?

DJ שמחפש שיר 30 שניות מאבד את הרחבה. DJ מאורגן מוצא כל שיר **מיד**.

ספרייה מסודרת = ביטחון על הבמה.

## ייבוא מוזיקה

### פורמטים נתמכים

| פורמט | איכות | גודל | מומלץ? |
|--------|--------|------|---------|
| WAV | מקסימלי | גדול מאוד | למפיקים |
| AIFF | מקסימלי | גדול מאוד | למפיקים |
| FLAC | Lossless | בינוני | ✅ הכי טוב |
| MP3 320kbps | מצוין | קטן | ✅ סטנדרט |
| MP3 128kbps | בינוני | קטן מאוד | ❌ לא לDJ |
| AAC/M4A | טוב | קטן | בסדר |

> **כלל:** לעולם אל תנגן MP3 מתחת ל-256kbps באירוע. הקהל ישמע את ההבדל ברמקולים חזקים.

### איך מייבאים?
1. **גרור ושחרר** — מתיקיית המוזיקה ישירות ל-Rekordbox
2. **File → Import** — ייבוא מתיקייה שלמה
3. **iTunes Integration** — ייבוא מספריית iTunes/Apple Music

## ניתוח אוטומטי

כשמייבאים שיר, Rekordbox מנתח אוטומטית:

### BPM (Beats Per Minute)
- מזהה את הקצב של השיר
- **דיוק**: 95%+ לאלקטרונית, 85%+ לשירים עם נגינה חיה
- **תמיד בדוק ידנית** שירים שנשמעים לא נכון

### Key (מפתח מוזיקלי)
- מזהה את הסולם — למשל: 8A, 11B, 2A
- משתמש במערכת Open Key או Camelot
- **חשוב ל-Harmonic Mixing** — מעברים שנשמעים נכון

### Waveform (גל צורה)
- מראה ויזואלית את מבנה השיר
- **צבעים** מציינים תדרים: כחול=בס, ירוק=אמצע, אדום=גבוהים
- עוזר לזהות Drops, Breakdowns, ו-Intros

## פלייליסטים

### סוגי פלייליסטים

**לפי ז'אנר:**
- House // Techno // Hip-Hop // Pop

**לפי אנרגיה:**
- 🔵 Low Energy (Warm-up)
- 🟡 Medium Energy (Build)
- 🔴 High Energy (Peak Time)
- 🟣 After Hours (Wind Down)

**לפי אירוע:**
- חתונה — Slow // רחבה // מזרחי
- מועדון — Opening // Peak // Closing
- בר — Background // Upbeat

### Intelligent Playlists

Rekordbox מאפשר ליצור פלייליסטים חכמים שמתעדכנים אוטומטית:
- "כל השירים 124-128 BPM עם דירוג 4+ כוכבים"
- "House שנוסף בחודש האחרון"

## דירוג ותיוג

### דירוג כוכבים (1-5)
- ⭐ = אולי אנגן פעם
- ⭐⭐ = שיר שאני מכיר
- ⭐⭐⭐ = שיר טוב שאני מנגן לפעמים
- ⭐⭐⭐⭐ = שיר מצוין — תמיד עובד
- ⭐⭐⭐⭐⭐ = המתנה שלי — Secret Weapon

### My Tag (תגיות אישיות)
- "Banger" — שירים שגורמים לאנשים לקפוץ
- "Transition" — שירים טובים למעברים
- "Request" — בקשות נפוצות

## סיכום: זרימת עבודה מומלצת

1. **ייבא** מוזיקה חדשה פעם בשבוע
2. **בדוק** BPM ו-Key שהניתוח נכון
3. **דרג** כל שיר אחרי שמאזין (1-5 כוכבים)
4. **סדר** בפלייליסטים לפי ז'אנר + אנרגיה
5. **מחק** שירים שלא מנגן — ספרייה קטנה ומדויקת > ספרייה ענקית ומבולגנת
