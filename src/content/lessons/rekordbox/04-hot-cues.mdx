# Hot Cues ו-Memory Cues — אסטרטגיית סימון מקצועית

## מה ההבדל?

| תכונה | Hot Cue | Memory Cue |
|--------|---------|------------|
| **לחיצה** | קופץ **ומנגן** | קופץ **ועוצר** |
| **כמה?** | 8 ב-Rekordbox, 8 ב-FLX4 | ללא הגבלה |
| **צבעים** | 16 צבעים | אדום/כתום |
| **CDJ תאימות** | CDJ-2000NXS2+ | כל CDJ (גם ישנים) |
| **שימוש עיקרי** | ביצוע חי — קפיצות מהירות | סימון מבנה, הכנה |
| **Quantize** | מומלץ ON | לא רלוונטי |

> **כלל אצבע:** Hot Cues ל**ביצוע** (קפיצות חיות). Memory Cues ל**הכנה** (סימון מבנה).

---

## מערכת Color Coding — 8 צבעים, 8 משמעויות

**עקביות היא הכל.** כשכל שיר בספרייה שלך מסומן **באותה שיטת צבעים**, אתה יכול לטעון כל שיר ולדעת בדיוק איפה כל דבר — **בלי לשמוע אותו קודם**.

### המערכת המומלצת

| Pad | שם | צבע | סמל | מה זה |
|-----|-----|------|------|--------|
| **1** | Intro | 🟢 ירוק | ▶ | **תחילת השיר** — הביט הראשון |
| **2** | Vocal/Verse | 🔵 כחול | ♪ | **תחילת ווקאל** או Verse |
| **3** | Build-Up | 🟡 צהוב | ⬆ | **תחילת Build-Up** — מתח עולה |
| **4** | Drop | 🔴 אדום | 💥 | **ה-Drop!** — הרגע הכי חזק |
| **5** | Mix-In | 🟠 כתום | ↗ | **נקודת כניסה** — איפה להתחיל מעבר |
| **6** | Mix-Out | 🟣 סגול | ↘ | **נקודת יציאה** — איפה להתחיל מעבר |
| **7** | Break | ⚪ לבן | ⏸ | **Breakdown** — חלק שקט |
| **8** | Special | 🟤 חום | ★ | **אלמנט מיוחד** — Acapella, נקודה ייחודית |

## Cue Workflow — איך לסמן שיר חדש

### תהליך (5-7 דקות לשיר)

```
שלב 1: 🎧 האזנה ראשונה — שמע את כל השיר (3-5 דק')
        └ רשום מנטלית: איפה ה-Drop? איפה ה-Build?
             ↓
שלב 2: ✅ הפעל Quantize (חובה!)
             ↓
שלב 3: 🟢 Cue 1 — Intro (ביט ראשון)
             ↓
שלב 4: 🔴 Cue 4 — Drop (הרגע הכי חזק)
             ↓
שלב 5: 🟠 Cue 5 — Mix-In (8-16 beats לפני ה-Drop)
             ↓
שלב 6: 🟣 Cue 6 — Mix-Out (תחילת ה-Outro)
             ↓
שלב 7: 🔵🟡⚪🟤 — הוסף פרטים: Vocal, Build, Break, Special
             ↓
שלב 8: ▶ נגן מתחילה ובדוק — כל Cue נופל בדיוק?
```

> **חשוב:** סמן **לפחות 4 Cues** בכל שיר (Intro, Drop, Mix-In, Mix-Out). את השאר — הוסף כשמכיר את השיר יותר טוב.

## שימוש ב-Cues בביצוע חי

### קפיצה ל-Drop

שיר B טעון ומוכן. במקום לנגן מההתחלה:
1. לחץ **Pad 4 (אדום)** — השיר קופץ ישר ל-Drop
2. הכנס עם EQ או Filter
3. **BAM** — ה-Drop פוגע מיד

### Loop על Mix-Out

1. שיר A מנגן → מגיע ל-**Pad 6 (סגול)** = Mix-Out zone
2. הפעל **Loop 8 beats** — שיר A חוזר על עצמו
3. הכנס שיר B
4. שחרר Loop כשהמעבר מוכן

### הקלטת Remix חי

1. נגן שיר A
2. קפוץ ל-**Pad 2 (כחול)** של שיר B — רק הווקאל
3. שלב ווקאל ב ועל Beat של A
4. **Instant Remix** — שני שירים → שיר חדש!

## Memory Cues — מתי וכיצד

### מתי להשתמש

- **CDJ-ים ישנים** — שלא תומכים ב-Hot Cues רבים
- **סימון מבנה** — "מכאן Verse 2", "פה השינוי"
- **נקודת עצירה** — השיר עוצר ומחכה ש-DJ ילחץ Play

### Memory Cue vs Hot Cue — דוגמה מעשית

```
Memory Cues (סימון מבנה):
├── MC1: "Intro Start" (0:00)
├── MC2: "Verse 1" (0:32)
├── MC3: "Chorus" (1:04)
├── MC4: "Verse 2" (1:36)
├── MC5: "Breakdown" (2:08)
├── MC6: "Drop" (2:40)
└── MC7: "Outro" (3:12)

Hot Cues (ביצוע):
├── HC1: 🟢 Intro     ← קפיצה להתחלה
├── HC4: 🔴 Drop      ← קפיצה ל-Drop
├── HC5: 🟠 Mix-In    ← נקודת כניסה
└── HC6: 🟣 Mix-Out   ← נקודת יציאה
```

## ייצוא Cues ל-USB

כל ה-Cues שהגדרת ב-Rekordbox **מיוצאים עם השיר:**
1. הגדר Cues ב-Rekordbox (בבית)
2. ייצא פלייליסט ל-USB
3. חבר USB ל-CDJ-3000 במועדון
4. **כל ה-Cues זמינים מיד** — אותם צבעים, אותן נקודות

---

## טיפים מקצועיים

| טיפ | למה |
|------|------|
| **עקביות** | אותה שיטת צבעים **בכל שיר** — תמיד |
| **Quantize ON** | Cue בדיוק על ביט — דיוק מושלם |
| **בדוק לפני אירוע** | עבור על כל שיר → ודא Cues נכונים |
| **לא מוחקים** | Cue מיותר עדיף על Cue חסר |
| **הכנה בבית** | **אף פעם** לא מגדירים Cues על הבמה |
| **שם ל-Cue** | קליק ימני → שם ("Drop", "Mix Out") |

## תרגול

### תרגיל 1: סמן 5 שירים (25 דק')
1. בחר 5 שירים מהספרייה
2. סמן בכל אחד **לפחות 4 Cues** (ירוק, אדום, כתום, סגול)
3. ודא עקביות צבעים בין כל השירים

### תרגיל 2: קפיצת Drop (10 דק')
1. טען 2 שירים מסומנים
2. שיר A מנגן
3. לחץ **Pad 4** של שיר B → Drop קופץ מיד
4. תרגל מעבר: A → B Drop
