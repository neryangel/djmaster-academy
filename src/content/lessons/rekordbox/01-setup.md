# Rekordbox — התקנה והגדרות

## הורדה והתקנה

### היכן להוריד

**https://rekordbox.com** (גרסה חופשית או Premium)

### מה להוריד

- **Rekordbox (Free)** — סביר לחלוטין להתחלה
- **Rekordbox (Premium)** — יותר תכונות (optional)

### שלבי התקנה

**Windows/Mac:**
1. הורד את ה-Installer
2. פתח את ה-Installer
3. בחר מקום ההתקנה
4. המתן 2-3 דקות
5. הפעל את Rekordbox

## הגדרה ראשונית

### שלב 1: התחברות לחשבון

1. פתח את Rekordbox
2. לחץ על "Sign In"
3. בחר "Create New Account" או התחבר
4. השתמש בדוא"ל ותן סיסמה חזקה

### שלב 2: בחר את תיקיית המוזיקה

1. Settings → Music Folder
2. בחר את התיקייה שבה יש לך מוזיקה (MP3, FLAC, WAV)
3. Rekordbox תחפש ותייבא את כל הקבצים

### שלב 3: בדוק את הציוד

1. Settings → Device Settings
2. בחר DDJ-FLX4 כ-Output
3. בדוק שקול יוצא מהרמקול

## ייבוא מוזיקה וניתוח אוטומטי

### מה זה ניתוח BPM?

Rekordbox מנתחת כל שיר:
- **BPM** — מהירות הטראק
- **Beat Grid** — קו שמציין כל beat
- **Waveform** — תצוגה ויזואלית

### תהליך הייבוא

**שלב 1: עדכן את תיקיית המוזיקה**
1. File → Music Folder
2. בחר את התיקייה שלך
3. לחץ על "Update" או "Scan"

**שלב 2: המתן לניתוח**
- 20-30 שניות למאה שיר
- תראה כן-כן עליון בזמן הניתוח

**שלב 3: בדוק את התוצאות**
1. בטל הצג רשימת השירים
2. בדוק שכל שיר יש BPM
3. אם חסר BPM — לחץ כפתור ימני וחזור על "Analyze"

## ניהול ספריה

### Playlists — ארגון לפי ז'אנר

1. עדיפות שמאל → "Playlist"
2. לחץ על "+" (ליד Playlist)
3. כתוב שם: "House 128 BPM"
4. לחץ "Create"

**להוסיף שירים:**
- גרור משיר מהרשימה הראשית לפלייליסט
- או בחר משיר, לחץ כן-כן, בחר "Add to Playlist"

### Tags & Color — ארגון מהיר

1. לחץ כן-כן על שיר
2. בחר "Edit → Tags"
3. כתוב תג: "Summer" או "Energy"
4. לחץ "Save"

**תגים מומלצים:**
- **Tempo:** "120", "128", "140"
- **Mood:** "Energy", "Chill", "Night"
- **Season:** "Summer", "Winter"

### Rating — דירוג שירים

1. לחץ כן-כן על שיר
2. בחר "Rating → ★★★★★"

דירוג גבוה = שיר טוב שאתה אוהב!

## Export Mode vs Performance Mode

| יכולת | Export Mode | Performance Mode |
|-------|-------------|------------------|
| **ניהול Playlist** | ✓ | - |
| **Grid Editing** | ✓ | - |
| **תגים & דירוג** | ✓ | - |
| **ביצוע** | - | ✓ |
| **Waveform** | לא | ✓ |
| **Mixing** | לא | ✓ |

**Export Mode** = ארגון מוזיקה
**Performance Mode** = ביצוע עם בקר

## הגדרה לביצוע עם DDJ-FLX4

### שלב 1: בחר את Output

**Preferences → Audio**
1. **Primary Output:** DDJ-FLX4 (MASTER)
2. **Headphones Output:** DDJ-FLX4 (HEADPHONES)

### שלב 2: עבור ל-Performance Mode

1. Ctrl+P (או Command+P ב-Mac)
2. בחר "DDJ-FLX4"
3. לחץ "OK"

### שלב 3: בדוק את הצליל

1. טען שיר
2. תנגן בבקר
3. אתה צריך לשמוע קול מהרמקול
