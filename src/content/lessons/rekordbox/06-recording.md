# הקלטה, ניתוח סטים, והפצה

## למה להקליט — 4 סיבות

| # | סיבה | הסבר |
|---|--------|------|
| 1 | **שיפור עצמי** | שומעים טעויות שלא שמענו חי |
| 2 | **Demo Mix** | כרטיס ביקור למארגנים ומועדונים |
| 3 | **נוכחות דיגיטלית** | בניית קהל אונליין |
| 4 | **תיעוד התקדמות** | לשמוע איך השתפרת ב-6 חודשים |

> **הכלל:** הקלט **כל תרגול**. גם אם זה לא מושלם. זו הדרך היחידה להשתפר.

---

## הגדרות הקלטה ב-Rekordbox

### Preferences → Recording

| הגדרה | ערך מומלץ | הסבר |
|--------|-----------|------|
| **פורמט** | WAV 24-bit | איכות מקסימלית |
| **Sample Rate** | 48 kHz | מקצועי |
| **תיקייה** | ~/DJ Recordings/ | מקום קבוע |

### פורמטי הקלטה

| פורמט | גודל/שעה | איכות | שימוש |
|--------|----------|-------|-------|
| **WAV 24-bit/48kHz** | ~1 GB | מקסימלי | שמירה + עריכה |
| **WAV 16-bit/44.1kHz** | ~600 MB | מצוין | אחסון |
| **MP3 320kbps** | ~140 MB | מצוין | שיתוף אינטרנט |

> **המלצה:** הקלט ב-**WAV**, ערוך, ואז ייצא ל-**MP3 320** לשיתוף.

---

## הקלטה — שלב אחר שלב

```
1. פתח Performance Mode
2. ודא: הגדרות הקלטה → WAV 24-bit
3. בדוק Levels — ודא צהוב, לא אדום
4. לחץ REC 🔴
5. נגן את הסט
6. סיים → לחץ STOP על REC
7. הקובץ נשמר אוטומטית
```

### טיפים קריטיים

| טיפ | למה |
|------|------|
| **בדוק Level לפני** | Clipping = הקלטה הרוסה |
| **אל תעצור** | טעית? המשך. ערוך אח"כ |
| **הכן Tracklist** | רשום שירים תוך כדי (או Rekordbox עושה) |
| **הקלט תרגולים** | גם אימונים. זה הדרך ללמוד |

## ניתוח סט מוקלט — מה לחפש

### צ'קליסט האזנה (30 דקות אחרי הסט)

```
□ מעברים — חלקים? שמעתי beat clash?
□ EQ — היו שני Bass Lines ביחד?
□ Timing — המעברים מדויקים או מוקדם/מאוחר?
□ בחירת שירים — הזרימה טבעית?
□ אנרגיה — בניתי מסע? או קפצתי?
□ אפקטים — שמתי יותר מדי? מעט מדי?
□ Levels — היו רגעים באדום (Clipping)?
```

### טבלת ניתוח מעברים

| # | מעבר | דקה | ציון (1-5) | מה עבד | מה לשפר |
|---|------|------|-----------|---------|---------|
| 1 | שיר A → B | 3:20 | ★★★★ | EQ חלק | Timing קצת מוקדם |
| 2 | שיר B → C | 6:45 | ★★★ | Filter טוב | Bass clash ב-7:00 |
| 3 | שיר C → D | 10:10 | ★★★★★ | מושלם! | — |

## עריכה בסיסית

### Rekordbox (מובנה)

- **Trim** — חיתוך התחלה וסוף
- **Normalize** — השוואת עוצמה

### Audacity (חינם — audacityteam.org)

**פעולות בסיסיות:**

| פעולה | מה | איך |
|--------|-----|------|
| **Trim** | הסרת שקט בהתחלה/סוף | Select + Delete |
| **Fade In** | כניסה הדרגתית | Select → Effect → Fade In |
| **Fade Out** | יציאה הדרגתית | Select → Effect → Fade Out |
| **Normalize** | איזון עוצמה | Select All → Effect → Normalize |
| **Export** | ייצוא ל-MP3 | File → Export as MP3 (320kbps) |

> **כלל:** אל תערוך יותר מדי. מיקס צריך להישמע **חי** — לא כמו הפקת אולפן.

## הפצה — איפה לשתף

### Mixcloud — מומלץ ראשון

| יתרון | הסבר |
|--------|-------|
| **חוקי** | רישיון ל-DJ Mixes עם מוזיקה מוגנת |
| **Content ID** | מזהה שירים אוטומטית |
| **ללא הגבלה** | אורך ומספר מיקסים |
| **חינמי** | Plan בסיסי ללא תשלום |
| **Tracklist** | נוצר אוטומטית |

### SoundCloud

| יתרון | חסרון |
|--------|--------|
| הכי פופולרי | Copyright Strikes על מיקסים |
| דף אמן מקצועי | חלק מהשירים מוסרים אוטומטית |
| אפשרות הורדה | Plan חינמי מוגבל (3 שעות) |

### YouTube

| יתרון | שימוש |
|--------|-------|
| חשיפה ענקית | DJ Set + ויזואליזציה |
| SEO טוב | אנשים מחפשים "House Mix 2025" |
| Live Streaming | שידור חי של סטים |

### Instagram / TikTok

| פלטפורמה | מה לפרסם | אורך |
|----------|----------|------|
| **Instagram Reel** | קליפ מהמעבר הכי טוב | 15-30 שניות |
| **TikTok** | רגע מרגש + תגובת קהל | 15-60 שניות |
| **Instagram Story** | Behind the scenes | 15 שניות |

## תרגול

### תרגיל 1: הקלט סט 15 דקות (20 דק')
1. בחר 5 שירים באותו BPM
2. **הפעל REC** → נגן → 4 מעברים
3. עצור הקלטה
4. האזן חזרה → מלא טבלת ניתוח

### תרגיל 2: ערוך ב-Audacity (15 דק')
1. פתח את ההקלטה ב-Audacity
2. Trim שקט מהתחלה/סוף
3. הוסף Fade In (3 שניות) ו-Fade Out (5 שניות)
4. Normalize
5. Export ל-MP3 320kbps

### תרגיל 3: העלה ל-Mixcloud (10 דק')
1. צור חשבון Mixcloud
2. העלה את המיקס
3. הוסף: Title, Description, Tags, Cover Art
4. פרסם → שתף בInstagram
