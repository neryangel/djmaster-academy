# Hot Cues ו-Memory Cues

## מה ההבדל?

| תכונה | Hot Cue | Memory Cue |
|--------|---------|------------|
| מפעיל את השיר? | ✅ קופץ ומנגן | ❌ קופץ ועוצר |
| כמה אפשר? | 8-16 | ללא הגבלה |
| עובד על CDJ ישן? | ❌ רק CDJ-2000NXS2+ | ✅ כל CDJ |
| שימוש עיקרי | ביצוע חי, קפיצות | סימון מבנה, הכנה |

## Hot Cues — הכלי הכי חשוב

### למה Hot Cues?
שיר ממוצע הוא 3-5 דקות. בלי סימון, אתה צריך לחפש כל פעם את ה-Drop, ה-Intro, ה-Outro. עם Hot Cues — **לחיצה אחת** ואתה שם.

### שיטת הסימון המקצועית

#### שיטת 4 Cues (מומלצת למתחילים)

| Cue | מיקום | צבע | שימוש |
|-----|--------|------|-------|
| **Cue 1** | תחילת ה-Intro | 🟢 ירוק | נקודת ההתחלה |
| **Cue 2** | תחילת ה-Drop 1 | 🔴 אדום | קפיצה ל-Drop |
| **Cue 3** | תחילת ה-Breakdown 2 | 🔵 כחול | נקודת מעבר |
| **Cue 4** | תחילת ה-Outro | 🟠 כתום | נקודת Mix-Out |

#### שיטת 8 Cues (מתקדמת)

| Cue | מיקום |
|-----|--------|
| 1 | Intro |
| 2 | Verse 1 / Build 1 |
| 3 | Drop 1 |
| 4 | Breakdown |
| 5 | Build 2 |
| 6 | Drop 2 |
| 7 | Outro Start |
| 8 | נקודת Mix-Out מדויקת |

### איך להגדיר ב-Rekordbox

1. **פתח שיר** בPlayer
2. **הפעל Quantize** (חובה — מבטיח דיוק)
3. **נגן** ועצור בנקודה הרצויה
4. **לחץ** על Pad ריק — Cue נשמר
5. **צבע** — לחץ ימני על ה-Cue → בחר צבע
6. **שם** — אפשר לתת שם (למשל "Drop", "Mix Out")

### כלל הזהב
> **הגדר Cues בבית, לא על הבמה.** הכנה מסודרת = ביצוע בטוח.

## Memory Cues — סימון מבנה

### למה Memory Cues?
Memory Cues שימושיים בעיקר כש:
- מנגנים על **CDJ ישנים** (שלא תומכים ב-Hot Cues רבים)
- רוצים **לסמן מבנה** בלי "לבזבז" Hot Cues
- צריכים **נקודת עצירה** מדויקת (Memory Cue עוצר את השיר)

### שימוש מעשי
1. Memory Cue בתחילת ה-Intro — השיר עוצר כאן ומחכה ש-DJ יפעיל
2. Memory Cue בתחילת Outro — תזכורת "כאן מתחיל לערבב"

## ייצוא ל-USB

Cues נשמרים ב-Rekordbox ו**מיוצאים עם השיר ל-USB**:
1. הגדר את כל ה-Cues ב-Rekordbox
2. ייצא את הפלייליסט ל-USB
3. חבר את ה-USB ל-CDJ
4. כל ה-Cues זמינים מיד

## טיפים מקצועיים

1. **עקביות**: השתמש תמיד באותה שיטת צבעים. אם ירוק = Intro, תשמור על זה בכל שיר
2. **Quantize ON**: תמיד כשמגדירים Cues — מבטיח שהCue נופל בדיוק על Beat
3. **בדיקת Cues**: לפני אירוע, עברום על כל השירים וודא שכל Cue בנקודה הנכונה
4. **לא מוחקים Cues**: עדיף Cue מיותר מאשר Cue חסר
