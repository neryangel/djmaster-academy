# 🛠️ 02-tools — כלים אינטראקטיביים

> 3 כלים HTML בעלי תכן עצמאי לשימוש ישיר בדפדפן

## סקירה

מודול הכלים מחזיק יישומונים HTML שונים שאינם דורשים שרת או התקנה. כל כלי הוא קובץ עצמאי שפתוח בדפדפן ועובד מיד (Plug & Play). אלו lead magnets חזקים בטיר Free של הפלטפורמה.

## כלים בודול זה

| כלי | תיאור | קובץ | גודל |
|-----|-------|------|------|
| **BPM Calculator** | מחשבון BPM עם Tap Tempo, המרה לפורמטים, מטרונום | `bpm-calculator/index.html` | 22 KB |
| **Harmonic Wheel** | גלגל Camelot אינטראקטיבי עם matching צבעים | `harmonic-wheel/index.html` | 36 KB |
| **Set Planner** | מתכנן סט DJ עם ניהול טראקים, עלייה אנרגיה | `set-planner/index.html` | 46 KB |

## Quick Start

פתח את הכלים ישירות בדפדפן (לא צריך שרת):

```bash
# בדפדפן:
file:///path/to/bpm-calculator/index.html
file:///path/to/harmonic-wheel/index.html
file:///path/to/set-planner/index.html
```

או פשוט לחץ פעמיים על הקובץ.
